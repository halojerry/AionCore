# 学术雷达纯提示词版（小白友好版）V3.0 - 专业严谨版

## 简介

学术雷达纯提示词版是一个专为学术研究设计的智能报告生成工具，**零配置、零依赖、零门槛**。

### 核心特点

- **零配置**：无需API Key、无需安装任何环境、无需配置参数
- **小白友好**：会打字就能用，专业术语尽量少用
- **纯提示词驱动**：所有功能都在提示词里，Agent直接执行
- **深度专业**：报告详实严谨，适合学术研究者深度阅读
- **智能交互**：支持自然语言配置，智能追问获取所需信息

### 版本选择

本工具提供两个版本：

1. **V2.0 基础版** (`prompts_v2.md`)
   - 适合快速浏览、获取学术动态
   - 报告结构简洁，重点突出
   - 适合初学者和时间有限的用户

2. **V3.0 专业版** (`prompts_v3_professional.md`) - **推荐**
   - 适合深度研究、学术分析
   - 报告详实严谨，分析深入
   - 包含热点总结、深度论文解析、趋势分析、选题建议
   - 适合有一定研究基础的用户

### 使用方法

#### 方法1：一键生成报告（推荐）

直接发送你的研究信息，Agent会自动识别并生成报告：

```
我是张三，北京大学物理学院的博士生，研究兴趣是黑洞物理和量子引力。核心关键词是black hole, quantum gravity, holography。请为我生成本周的学术雷达报告。
```

Agent会：
1. 自动识别你的研究信息
2. 搜索相关RSS源
3. 筛选高质量论文
4. 生成完整专业报告

#### 方法2：交互式配置

如果你不确定该怎么配置，可以直接说：

```
我想生成学术周报，但不知道怎么配置
```

Agent会引导你一步步完成配置。

#### 方法3：参考领域模板

发送「参考模板」，获取预置的领域配置模板，复制修改即可。

### 报告输出示例

报告将包含以下模块：

1. **执行摘要** - 本周监测概况、内容分布、质量分布
2. **领域热点核心摘要** - 研究脉络、关键突破、跨学科融合、方法论演进、潜在机会
3. **学术论文深度解析** - 每篇论文的完整元数据、深度评分、研究背景、方法创新、实验结果、学术价值
4. **研究趋势深度分析** - 来源分布、质量分布、主题聚类、时间维度、作者机构分析
5. **研究机会与选题建议** - 基于热点分析的个性化选题建议
6. **高质量文献归档** - 便捷查阅的核心论文列表
7. **下期关注建议** - 基于趋势预测的个性化关注方向

### 评分标准

本工具使用多维度评分机制（总分100分）：

| 维度 | 权重 | 说明 |
|------|------|------|
| 关键词匹配度 | 35% | 标题、摘要、作者信息的多维度匹配 |
| 学术影响力 | 25% | 期刊/会议影响因子、学术声誉 |
| 时效性 | 15% | 内容新鲜程度，优先本周发表 |
| 新颖性 | 15% | 创新性、突破性贡献评估 |
| 来源可靠性 | 10% | 信息源权威性、数据完整性 |

### 技术原理

本工具采用**纯提示词驱动**架构，无需：
- 调用外部API（如RSS解析API、LLM API等）
- 执行外部代码（如Python脚本、JS脚本等）
- 访问本地文件系统
- 配置任何环境变量

所有功能由当前Agent的能力直接承载，利用Agent的搜索、分析、总结能力完成任务。
