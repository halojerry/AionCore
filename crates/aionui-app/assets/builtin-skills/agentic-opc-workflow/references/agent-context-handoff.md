# Agent 上下文交接包

Agent 接不到好任务，通常不是模型不行，而是上下文交接太烂。

## 好的交接包包含

```markdown
# Agent Handoff

## 你是谁
{Agent 在这次任务中的角色，不是人格表演}

## 背景
{用户为什么要做这件事}

## 真实目标
{真正要解决的问题}

## 输入材料
{已有文档、访谈、数据、链接、观点}

## 任务边界
{本次只做什么，不做什么}

## 输出格式
{Markdown 表格 / 清单 / 报告 / JSON / 草稿}

## 判断标准
{什么样的输出算有用}

## 反例
{什么样的输出一看就跑偏}

## 需要停下来问人的地方
{不确定、风险高、需要价值判断时停止}
```

## 常见坏任务

- “帮我做个完整方案”
- “帮我分析一下这个方向”
- “帮我写一套工作流”

坏在：没有背景、没有边界、没有验收标准，Agent 只能猜。
