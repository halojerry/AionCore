# Human Gates — 人类判断门

Agentic 工作流不是越自动越好。真正可靠的工作流，会在关键节点让人重新接管判断。

## 必须设置 Human Gate 的节点

| 节点 | 为什么 |
|---|---|
| 方向判断 | 是否值得做由人承担后果 |
| 对外发布 | 涉及品牌和信任 |
| 收费承诺 | 涉及交易和交付责任 |
| 用户隐私 | 涉及合规和信任 |
| 数据解释 | AI 容易把相关性说成因果 |
| 转向 / 停止 | 这是价值取舍，不是执行任务 |

## Gate 模板

```markdown
## Human Gate：{节点名}
Agent 到这里必须停止，用户亲自回答：
1. 这个结果是否符合我的真实意图？
2. 关键证据是否足够？
3. 如果错了，代价我能不能接受？
4. 有没有更小的验证动作？
5. 是否需要停止或转向？
```

## 原则

Agent 可以帮你把材料摆上桌，但不能替你决定哪道菜该吃。
