# 证据阶梯

做认知反缴械时，先把 AI 输出分层。很多混乱来自把假设当事实，把推断当结论。

## 五层

### 1. 事实

可被直接查证的客观信息。

例：某产品上线时间、价格、用户数、融资公告。

### 2. 证据

支持某个判断的信息，可以强也可以弱。

例：用户访谈、转化数据、行业报告、付费记录。

### 3. 推断

基于事实和证据做出的解释。

例：用户愿意付费可能是因为现有工具太复杂。

### 4. 假设

目前还没有充分证据，但决定成立需要它为真。

例：一人公司创作者愿意为思维陪练付费。

### 5. 漂亮话

听起来正确，但无法指导行动，也没有证据。

例：AI 将重塑工作方式、用户需要更高效的解决方案、关键在于持续迭代。

## 报告里的标注

- 事实：要求给来源或可验证路径
- 证据：说明强弱和来源
- 推断：说明推理链
- 假设：说明如何验证
- 漂亮话：删除或改成具体判断
