# 判断边界：哪些能交给 AI，哪些不能

## 可以交给 AI 的事

- 搜集材料
- 整理上下文
- 提炼选项
- 找反例
- 生成草稿
- 模拟用户问题
- 做检查清单
- 形成初版执行计划

## 不能完全交给 AI 的事

- 价值取舍：什么更重要，牺牲什么
- 风险承担：亏钱、失去机会、影响关系的后果谁来扛
- 真实承诺：是否向用户收费、是否公开发布、是否签约
- 长期选择：做不做这个赛道、是否 all in、要不要转向
- 道德与法律边界：什么不能做

## 人类判断门

凡是涉及这些动作，必须设置 Human Gate：

- 对外发布重要内容
- 收钱 / 退款 / 承诺交付
- 法律、财务、医疗、投资相关判断
- 用户隐私和敏感数据处理
- 不可逆的产品或商业决策

Human Gate 的写法：

```markdown
## 人类判断门
在 Agent 输出后，用户必须亲自检查：
1. 这个建议是否符合我的真实目标？
2. 风险我是否愿意承担？
3. 有没有证据支持关键判断？
4. 有没有侵犯用户、客户或合作方利益？
5. 如果错了，我是否知道怎么止损？
```
