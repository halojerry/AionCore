## 概览
本规范指导小学语文习题的分层设计，确保习题难度递进，符合不同层次学生的学习需求，激发学习兴趣。

## 核心内容

### 1. 分层原则

**目标导向**：
- 基础题：面向全体学生，巩固基础知识
- 提高题：面向中等及以上学生，提升能力
- 拓展题：面向学有余力学生，培养创新思维

**难度递进**：
- 基础题：识记与理解层次
- 提高题：应用与分析层次
- 拓展题：综合与创造层次

**数量配置**：
- 基础题：3-5道
- 提高题：3-5道
- 拓展题：2-3道

### 2. 基础题设计标准

**能力要求**：
- 识记：记住生字新词、课文内容
- 理解：理解字词意思、课文主旨

**题型示例**：
1. 生字类：注音、组词、选字填空
2. 词语类：解释词语、近义词、反义词
3. 句子类：选词填空、照样子写句子
4. 内容类：选择题、填空题

**设计要点**：
- 直接对应课文内容
- 难度适中，大部分学生能独立完成
- 答案明确，易于检查
- 避免歧义和模糊表述

**示例（以《小蝌蚪找妈妈》为例）**：

**一、基础题**

1. 给下面的字注音。
   蝌（ ） 蚪（ ） 鲤（ ） 鳖（ ）

2. 组词语。
   河（ ） 江海（ ）
   池（ ） 荷叶（ ）

3. 选择正确的词语填空。
   （游来游去 跑来跑去）
   小蝌蚪在（ ），小白兔在草地上（ ）。

4. 填空。
   小蝌蚪先长出______，再长出______，______变短，最后变成青蛙。

5. 按课文内容填空。
   小蝌蚪的脑袋________，身子________，尾巴________。

### 3. 提高题设计标准

**能力要求**：
- 应用：运用所学知识解决问题
- 分析：分析课文结构和写作手法
- 联系：联系生活实际，拓展思维

**题型示例**：
1. 词语运用：用指定词语造句
2. 句子仿写：照样子写句子
3. 内容理解：理解含义深刻的句子
4. 逻辑推理：根据课文内容推理
5. 联想想象：根据情境展开想象

**设计要点**：
- 需要一定的思考和推理
- 答案不唯一，鼓励创新
- 贴近生活实际
- 培养思维能力

**示例（以《小蝌蚪找妈妈》为例）**：

**二、提高题**

1. 用"游来游去"写一句话。
   ___________________________________________

2. 照样子写句子。
   例：小蝌蚪游来游去。
   小鸟________来________去。
   蝴蝶________来________去。

3. 读句子，理解句子的意思。
   "小蝌蚪游哇游，过了几天，长出了两条后腿。"
   这句话告诉我们什么？
   ___________________________________________

4. 小蝌蚪在找妈妈的过程中，先后遇到了鲤鱼阿姨、乌龟、青蛙。
   你觉得小蝌蚪的心情是怎样的？为什么？
   ___________________________________________

5. 课文中有"大大的脑袋"、"长长的尾巴"这样的词语，你觉得好在哪里？
   ___________________________________________

### 4. 拓展题设计标准

**能力要求**：
- 创造：创意写作、创新表达
- 综合：综合运用多种能力
- 实践：观察、调查、动手操作
- 探究：探究性问题解决

**题型示例**：
1. 创意写作：续写故事、改写结局
2. 实践活动：观察日记、调查报告
3. 拓展阅读：推荐相关阅读材料
4. 跨学科：结合其他学科知识
5. 开放性问题：鼓励多角度思考

**设计要点**：
- 难度较大，适合学有余力的学生
- 答案开放，鼓励创新思维
- 贴近生活，培养实践能力
- 可以小组合作完成

**示例（以《小蝌蚪找妈妈》为例）**：

**三、拓展题**

1. 小蝌蚪最后找到了妈妈，心里一定很高兴。请你续写故事：小蝌蚪找到妈妈后，会说什么？会做什么？
   ___________________________________________

2. 你还知道哪些动物的生长过程？请查阅资料或询问爸爸妈妈，写下来。
   ___________________________________________

3. 观察生活中的一种小动物，写一篇观察日记。
   要求：
   （1）写出小动物的样子
   （2）写出小动物在做什么
   （3）写出你的感受
   ___________________________________________

4. 课文中的小蝌蚪很有耐心，找了很多次才找到妈妈。你在生活中有没有遇到过困难？是怎么解决的？写一写。
   ___________________________________________

5. 小组合作：制作一张小动物成长过程的手抄报。
   要求：
   （1）选择一种小动物
   （2）画出它的成长过程
   （3）配上简单的文字说明

### 5. 习题示例

**完整示例（以《春晓》为例）**：

**一、基础题**

1. 给下面的字注音。
   春（ ） 晓（ ） 眠（ ） 闻（ ）

2. 组词语。
   春（ ） 春风（ ）
   花（ ） 花朵（ ）

3. 选择正确的解释。
   "春眠不觉晓"中的"晓"意思是（ ）
   A. 早晨 B. 知道 C. 醒来

4. 默写古诗。
   《春晓》
   ________________，________________。
   ________________，________________。

5. 填空。
   这首诗的作者是______代的______。

**二、提高题**

1. 用"处处"写一句话。
   ___________________________________________

2. "夜来风雨声，花落知多少。"这两句诗表达了诗人怎样的情感？
   ___________________________________________

3. 你能用自己的话说说这两句诗的意思吗？
   "夜来风雨声，花落知多少。"
   ___________________________________________

4. 春天来了，你还看到哪些美丽的景象？写一写。
   ___________________________________________

5. 这首诗描写了春天的什么特点？
   ___________________________________________

**三、拓展题**

1. 你还知道哪些描写春天的古诗？写一首下来。
   ___________________________________________

2. 画一幅春天的画，配上诗句。

3. 观察春天，写一篇观察日记。

4. 小组合作：制作一本"春天的诗集"。

## 验证规则
- [ ] 包含三个层次：基础、提高、拓展
- [ ] 每个层次3-5道题
- [ ] 难度递进明显
- [ ] 题型多样，覆盖不同能力要求
- [ ] 贴近课文内容，联系生活实际

## 常见错误
- ❌ 层次不明显，题目难度混乱
- ❌ 题量过多或过少
- ❌ 脱离课文内容，难度过大
- ❌ 答案唯一，缺乏开放性
- ❌ 题型单一，缺乏趣味性
