# 教案结构规范

## 目录
1. 教案八大模块
2. 各科目核心素养维度对照
3. 教学过程时间分配
4. 教案格式要求

## 概览
本规范指导小学各科目教案的撰写，确保教案结构完整、符合2022版新课标要求。

## 核心内容

### 1. 教案八大模块

#### 1.1 基本信息
- 课题名称
- 年级/学期
- 课时安排（共几课时，本课为第几课时）
- 教材版本（部编人教版/北师大版/外研版等）
- 授课教师

#### 1.2 教学目标（基于2022版新课标核心素养）

**语文** — 四维度：
1. 文化自信：对中华文化的认同感和自豪感
2. 语言运用：积累语言经验，形成良好语感
3. 思维能力：直觉思维、形象思维、逻辑思维、辩证思维、创造思维
4. 审美创造：感受美、发现美、表现美、创造美

**数学** — 十一核心素养：
1. 数感、量感、符号意识
2. 运算能力、推理意识、推理能力
3. 几何直观、空间观念
4. 数据意识、数据观念
5. 模型意识、模型观念
6. 应用意识、创新意识

**英语** — 四维度：
1. 语言能力：感知与积累、习得与建构、表达与交流
2. 文化意识：比较与判断、调试与沟通、感悟与内化
3. 思维品质：观察与辨析、归纳与推断、批判与创新
4. 学习能力：乐学与善学、选择与调整、合作与探究

**科学** — 四维度：
1. 科学观念：对科学知识的理解和应用
2. 科学思维：模型建构、推理论证、创新思维
3. 探究实践：提出问题、设计方案、收集证据、解释结论
4. 态度责任：科学态度、社会责任

**道法** — 五维度：
1. 政治认同：国家认同、文化认同、制度认同
2. 道德修养：个人品德、家庭美德、社会公德
3. 法治观念：宪法意识、权利意识、义务意识
4. 健全人格：自尊自信、理性平和、积极向上
5. 责任意识：主人翁意识、担当精神、有序参与

#### 1.3 教学重难点
- 教学重点：本课最核心的知识/能力目标（1-2条）
- 教学难点：学生理解或掌握的困难之处（1-2条）

#### 1.4 教学准备
- 教师准备：教具、多媒体课件、实验器材等
- 学生准备：预习内容、学具、课前调查等

#### 1.5 教学过程（核心模块，分环节标注时间）

| 环节 | 时间 | 教师活动 | 学生活动 | 设计意图 |
|------|------|---------|---------|---------|
| 导入 | 3-5min | 创设情境，激发兴趣 | 感知、思考、回应 | 引出课题 |
| 新授 | 15-20min | 分步讲解，引导探究 | 听讲、实践、讨论 | 学习新知 |
| 巩固 | 5-8min | 组织练习，巡视指导 | 独立练习、合作交流 | 内化知识 |
| 小结 | 2-3min | 引导总结，点明要点 | 归纳、回顾 | 梳理知识 |
| 作业 | 1-2min | 布置作业，明确要求 | 记录作业 | 延伸学习 |

#### 1.6 板书设计
- 主板书：知识框架、关键概念、核心公式/句型
- 副板书：辅助说明、例题演算、临时记录
- 提供板书布局示意图

#### 1.7 作业布置
- 必做题：巩固基础知识，面向全体学生
- 选做题：拓展提升，面向学有余力学生
- 实践作业：联系生活，培养应用能力

#### 1.8 教学反思
- 成功之处：教学目标达成情况、亮点环节
- 不足之处：需要改进的地方
- 改进措施：具体改进方案

### 2. 教学过程时间分配

| 环节 | 时间 | 说明 |
|------|------|------|
| 导入环节 | 3-5分钟 | 含教师语言和预期学生回应 |
| 新授环节 | 15-20分钟 | 分步骤，每步骤标注小标题 |
| 巩固环节 | 5-8分钟 | 练习+反馈 |
| 小结 | 2-3分钟 | 师生共同总结 |
| 作业布置 | 1-2分钟 | 必做+选做 |

### 3. 教案格式要求

- 教学目标严格使用对应科目课标维度，不使用旧版"知识与技能、过程与方法、情感态度价值观"三维目标
- 教学过程中用「师：」「生：」标注对话，体现师生互动
- 每个环节标注时间，环节之间逻辑清晰
- 板书设计提供示意图，方便教师课堂实施
- 作业分层设计，兼顾不同层次学生
