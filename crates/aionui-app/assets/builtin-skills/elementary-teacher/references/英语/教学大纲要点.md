# 小学英语教学大纲要点

## 目录
1. 课程目标
2. 内容标准（按学段）
3. 教学建议
4. 评价建议

## 概览
本文档提炼小学英语教学大纲的核心要点，为教案设计、讲课稿撰写提供指导和依据。依据《义务教育英语课程标准（2022年版）》。

## 核心内容

### 1. 课程目标

#### 1.1 核心素养

- **语言能力**：感知与积累、习得与建构、表达与交流
- **文化意识**：比较与判断、调适与沟通、感悟与内化
- **思维品质**：观察与辨析、归纳与推断、批判与创新
- **学习能力**：乐学与善学、选择与调整、合作与探究

#### 1.2 分级目标

小学阶段对应**一级**（3-4年级）和**二级**（5-6年级）。

**一级目标（3-4年级）**：
- 对英语有好奇心，喜欢听他人说英语
- 能根据教师的简单指令做动作、做游戏、做事情
- 能做简单的角色表演
- 能唱简单的英文歌曲，说简单的英语歌谣
- 能理解简单的图片和故事
- 能交流简单的个人信息，表达简单的感觉和情感
- 能书写字母和简单的单词
- 对英语学习中接触的外国文化习俗感兴趣

**二级目标（5-6年级）**：
- 对继续学习英语有兴趣
- 能用简单的英语互致问候，交换简单的信息
- 能就日常生活话题作简短叙述
- 能在图片的帮助下听懂、读懂简单的小故事
- 能在图片和提示词的帮助下描述或讲述简单的小故事
- 能根据图片、词语或例句的提示写出简短的描述
- 乐于了解外国文化和习俗

### 2. 内容标准

#### 2.1 语言技能

**听做（一级）**：
- 能根据听到的词语识别或指认图片或实物
- 能听懂课堂简短的指令并做出相应的反应
- 能根据指令做事情，如指图片、涂颜色、画图、做动作等
- 能在图片和动作的提示下听懂简单的小故事

**听（二级）**：
- 能借助图片、图像、手势听懂简单的话语或录音材料
- 能听懂简单的配图小故事
- 能听懂课堂活动中简单的提问
- 能听懂常用指令和要求并做出适当反应

**说唱（一级）**：
- 能根据录音模仿说话
- 能相互致以简单的问候
- 能相互交流简单的个人信息
- 能表达简单的情感和感觉
- 能根据表演猜测意思、说出词语
- 能学唱英语儿童歌曲和歌谣15首左右
- 能根据图、文说出单词或短句

**说（二级）**：
- 能在口头表达中做到发音清楚，语调基本达意
- 能就所熟悉的个人和家庭情况进行简短对话
- 能运用一些最常用的日常套语
- 能就日常生活话题作简短叙述
- 能在教师的帮助和图片的提示下描述或讲述简单的小故事

**读写（一级）**：
- 能看图识词
- 能在指认物体的前提下认读所学词语
- 能在图片的帮助下读懂简单的小故事
- 能正确书写字母和单词
- 能模仿范例写词句

**读写（二级）**：
- 能认读所学词语；能根据拼读规律读出简单的单词
- 能读懂教材中简短的要求或指令
- 能看懂贺卡等所表达的简单信息
- 能借助图片读懂简单的故事或小短文
- 能正确使用大小写字母和标点符号
- 能写出简单的问候语和祝福语
- 能根据图片、词语或例句的提示，写出简短的语句

#### 2.2 语言知识

**语音**：
- 一级：正确读出26个字母；了解简单的拼读规律；了解重音和升降调
- 二级：了解字母名称的读音；了解拼读规律；用正确的语音语调朗读

**词汇**：
- 一级：学习与个人、家庭、学校生活相关的词汇（约200词）
- 二级：学习有关本级话题范围的词汇（约500-600词+习惯用语）

**语法**：
- 一级：名词单复数、人称代词、常用介词、一般现在时、现在进行时
- 二级：上述基础上增加：形容词比较级、一般过去时、一般将来时、There be结构

**功能**：问候、介绍、告别、感谢、道歉、请求、邀请、建议、购物、问路、时间、天气等

**话题**：个人情况、家庭朋友、学校生活、日常活动、饮食、天气、节日、动植物等

#### 2.3 情感态度

- 保持学习英语的兴趣和积极性
- 敢于开口说英语，不怕出错
- 乐于参与课堂活动，积极与他人合作
- 遇到困难时不放弃，主动寻求帮助

#### 2.4 学习策略

- 在词语与相应事物之间建立联想
- 积极运用所学英语进行表达和交流
- 注意观察生活或媒体中使用的简单英语
- 能初步借助简单的工具书学习英语

#### 2.5 文化意识

- 知道英语中最简单的称谓语、问候语和告别语
- 对一般的赞扬、请求、道歉等做出适当的反应
- 知道主要英语国家的首都和国旗
- 了解主要英语国家的重要标志物
- 了解主要英语国家的重要节假日
- 在学习和日常交际中，能初步注意到中外文化异同

### 3. 教学建议

- 创设真实语境，让学生在语境中学习语言
- 采用PPP模式（Presentation-Practice-Production）
- 听说领先，读写跟上
- 使用TPR（全身反应法）进行教学，尤其在低年级
- 利用歌曲、歌谣、游戏激发兴趣
- 重视语音教学和拼读规律
- 任务型教学：让学生在完成任务中运用语言
- 分层设计活动，照顾不同水平学生

### 4. 评价建议

- 形成性评价为主，终结性评价为辅
- 评价内容涵盖语言技能、语言知识、情感态度、学习策略和文化意识
- 评价形式多样化：口语测试、听力测试、角色表演、作品展示
- 关注学生进步，以鼓励性评价为主
