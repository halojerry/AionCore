# 投稿前引用体检清单（citation_checklist）

> 配合 `scripts/verify_refs.py`（真实性/一致性机读核验）与 `scripts/doi_to_any.py`（格式生成）使用。
> 勾选制，逐项过；任一项不达标即回到对应工具修复。诚实原则见 CONVENTIONS §4：不臆造、区分已验证与推测。

## A. 真实性（硬门槛，零容忍）
- [ ] 每条引用都能在 Crossref `/works/{doi}` 命中（HTTP 200）。跑 `verify_refs.py --file dois.txt`。
- [ ] `summary.high_severity_errors == 0`（无查不到的 DOI）。任何 high error 必须删除或换源，不得带病投稿。
- [ ] Crossref 与 OpenAlex 标题/年份一致（无 `title 不一致`/`年份不一致` warning），不一致项人工定夺。
- [ ] DOI 指向的就是正文所述工作（标题、作者、年份三对一致），杜绝张冠李戴。
- [ ] 预印本（arXiv/bioRxiv）已注明"未经同行评审"，且若已正式发表则换成正式版 DOI。

## B. 关联度与正文匹配
- [ ] 每个 `\cite` 都能指回它支撑的具体句子/论点（借鉴 Anthropic Citations 的"可定位"标准）。
- [ ] 正文声称的第三方引用关系（"A 引用了 B"）已用 `verify_citation_edge.py <A> <B>` 实证：取 `confirmed`；得 `not_in_open_index` 须人工查全文/WoS/Scopus 后再断言（开放索引未覆盖≠未引用）；得 `unknown` 不下结论。
- [ ] 无"凑数引用"：删掉与论点无直接关系的来源。
- [ ] 综述 vs 原始工作用对地方：方法/数据归原始论文，背景概览才引综述。

## C. 覆盖度自检（中外 / SOTA / 经典）
| 维度 | 要求 | 检查方式 |
|---|---|---|
| 领域经典/奠基 | 至少覆盖该问题的奠基工作 | 人工 + OpenAlex `cited_by_count:>` 高被引扫描 |
| 最新进展（近 1–2 年） | 至少 N 篇近 2 年 | `summary.missing_recent_2y == false` |
| 直接对比 / SOTA baseline | 所有正文比较过的方法都已引 | 对照实验表逐行核 |
| 中外占比 | 按 venue 合理（见下表） | `summary.cn_ratio` |
| 数据集 / 工具来源 | 用到的数据集、代码库均已引原始出处 | 人工 |

## D. 数量与健康度
- [ ] 总量与论文篇幅/类型相称（短文 ~20–40，长文/综述更多），非越多越好。
- [ ] 自引率合理：`summary.self_citation_rate` 一般 < 0.2（综述可略高），过高需删减。跑时带 `--self-author 姓氏`。
- [ ] 无重复条目（同一工作多个 DOI/版本混入）。
- [ ] 来源权威：慎引掠夺性期刊/无同行评审来源；可疑来源上报。

## E. 格式与机械正确
- [ ] 已按目标 venue 选定样式（GB/T 7714 顺序编码 / IEEE / APA …），用 `doi_to_any.py --format` 生成。
- [ ] citekey 已 pin（Better BibTeX），避免元数据变动导致 `\cite` 失效。
- [ ] 文献类型标识码正确（`[J]/[C]/[M]/[D]/[EB/OL]`，GB/T 7714）。
- [ ] 作者 >3 处理一致（GB/T 7714 取前 3 加"等"）。
- [ ] LaTeX 后端与模板要求一致（顶会锁 bibtex+.bst 时勿擅自换 biber）。

## 中外占比建议（按 venue）
| 投稿去向 | 中文文献占比建议 | 说明 |
|---|---|---|
| 中文核心期刊 | 视主题，通常 ≥ 30–50% | 需体现对国内同行工作的了解 |
| 国际期刊 / 顶会（英文） | 以英文为主，中文按需 | 引中文工作时给出英文译名/DOI |
| 学位论文 | 中外兼顾、覆盖全面 | 经典+最新+国内外都要 |

> 占比无统一硬标准，以"覆盖该领域真正重要的工作、不因语言遗漏关键文献"为准则。

## 被引阈值分档表（按文献年龄判代表性）
> 用途：判断一篇被引数 = 代表性强还是凑数。被引随时间累积，必须按"出版年龄"分档，不能拿新论文和老论文比绝对值。
> 阈值为经验参考量级（领域差异大：CS/生医引用快，数学/人文慢），仅作粗筛，不替代人工判断。

| 出版年龄 | 低（代表性弱/慎引为唯一支撑） | 中（正常） | 高（代表性强/领域重要） |
|---|---|---|---|
| ≤ 1 年（新） | 0（正常，太新） | 1–10 | > 10（已显影响力） |
| 2–3 年 | < 5 | 5–50 | > 50 |
| 4–6 年 | < 15 | 15–150 | > 150 |
| 7–10 年 | < 30 | 30–300 | > 300（准经典） |
| > 10 年 | < 50（若仍被引说明有长尾） | 50–500 | > 500（领域经典/奠基） |

判读规则：
- 新论文低被引不扣分（时间不够）；老论文长期零被引则代表性存疑，避免当唯一支撑。
- `verify_refs.py` 已对"较旧且被引偏低（>6 年且 <5 次）"自动产 warning，对照本表确认。
- "高被引"不等于"对本文最相关"——相关度永远优先于被引数。
- 跨库被引数口径不同：Crossref `is-referenced-by-count` / OpenAlex `cited_by_count` / OpenCitations 仅开放 DOI-DOI，普遍偏低，别混用排名。
