## 0. 当届规则确认闸门（rules_checklist —— 流程第一步必产的工件）

SKILL 流程第一步「先定位赛事」不只是口头确认，要**落盘一个可勾选工件** `material_checklist.md`（或单独 `rules_checklist.md`），把"我已对齐当届规则"从一句口头承诺升级为可核查的勾选记录。模板：

```markdown
# <赛事名> <当届> 规则确认清单（rules_checklist）
> 规则压缩包来源：<官网下载链接 / 文件名>　｜　下载日期：<YYYY-MM-DD>　｜　确认人：<>

- [ ] **赛事/组别**：确认是 <数模国赛/美赛 ∣ 创新大赛创意组/创业组 ∣ 大挑/小挑 ∣ 大创创新/创业训练/创业实践>，评审重点已对齐
- [ ] **当届重点领域**：本届领域分类为 <___>，本项目归属 <___>
- [ ] **页数/篇幅上限**：<___ 页/字>，含/不含附录代码 = <___>
- [ ] **提交物清单**：需提交 <申报书/BP/路演PPT/答辩/视频/…>，格式 <PDF/PPTX/…>
- [ ] **预算封顶/科目限制**：<总额上限 ___；设备费占比限制 ___>（若有）
- [ ] **AI 使用规定**：本届是否要求声明 AI 使用？<未用须声明/用了须标注+提交详情/无规定>
- [ ] **截止时间与评审阶段**：初评 <___>，终审/答辩 <___>
- [ ] **组别名称未套旧届**：已核对组别名是当届叫法（anti_patterns D2）

> 任一项打不了勾 = 还没下当届规则，**不要动笔**。本清单随材料一起交付前复核。
```

> 为什么是工件而非口头：anti_patterns D2「套用旧届模板/组别名称过时」是高频低级出局点；E1「没确认赛事就动笔」等于答非所问。一张勾选清单把这两类风险前置拦截。

## 1. 数学建模国赛 CUMCM

- **评审重点优先级**：摘要（单独一页、决定初筛）＞结果＞模型与求解。
- **论文必含**：模型假设、模型建立与求解、结果检验、**灵敏度分析**、优缺点、附录完整可复现代码。
- **评审机制**：两级评审（赛区初评 → 全国终评）。
- **AI 使用规定（2025 年试行，自 2025-09-01 起；last_checked 2026-06）**：允许但非必须用 AI 工具（含 LLM/生成式/代码辅助/智能数学软件）；核心建模与分析须参赛队独立完成。
  - **未用**：须在参考文献后声明"本参赛队未使用任何 AI 工具"。
  - **用了**：①正文标注 AI 生成内容并在参考文献列工具信息（名称/版本/机构/日期）；②提交 PDF「AI 工具使用详情」（工具、用途环节、关键提示词与回复、采纳与人工修改情况）。
  - 不符标注/说明要求 → **取消评奖资格**。官方源 dxs.moe.gov.cn（2026-06 核实）。

## 2. 数学建模美赛 MCM/ICM

- **Summary Sheet**：必须是第一页，评委给极高权重（常靠它拉开差距）；忌问题重述/复制引言，**最后写**。
- **页数硬上限**：全篇 **25 页**（含摘要 + 正文 + 参考文献 + 目录 + 附录 + 代码）；**唯一例外是必填的 "Report on Use of AI"，不计页**。
- **赛制**：99 小时、三人队。
- **奖项分级**（无及格线，按完成度评级）：Successful Participant ＜ Honorable Mention ＜ Meritorious ＜ Finalist ＜ Outstanding。
- **题型**：MCM A 连续 / B 离散 / C 数据；ICM D 运筹 / E 可持续 / F 政策。

## 3. 统计建模大赛

- **评审重点**：真实数据来源、变量解释、统计推断严谨、现实意义 ＞ 算法炫技。
- **常用范式**：多模型对比；统计推断给置信区间/检验（复用 m06 result-analysis 的 `code_assets/stats_tests.py`）。
- **出局点**：数据来源存疑、变量无解释、堆花哨模型却推断不严谨（anti_patterns C2）。

## 4. 中国国际大学生创新大赛（原"互联网+"）

- **更名**：2024 年起由"互联网+大学生创新创业大赛"更名为"中国国际大学生创新大赛"。套旧名即过时（anti_patterns D2）。
- **高教主赛道组别**：按学历分**本科生组 / 研究生组**，每组下设**创意组**（有创意、未注册公司或注册≤3年未盈利）和**创业组**（已注册公司）。**2024+ 已淘汰旧的"初创组/成长组/师生共创组"**。
- **当届重点领域导向**（逐年微调，写前对齐）：新工科、新医科、新农科、新文科、人工智能+、低空经济、生物技术、量子科技、新能源、新材料等。
- **提交物**：商业计划书 + 路演 PPT + 答辩。
- **评审维度（公开口径）**：创新性、团队、商业/社会价值、带动就业、可行性、引领性、（部分赛道）国际化。【确切权重待核查当届】
- **官方入口**：https://cy.ncss.cn/ （评审规则与学生操作手册以压缩包形式在站内"资料下载"逐年发布）。

## 5. 挑战杯（两个并列主体竞赛，先确认是哪一杯）

- **"大挑" = 全国大学生课外学术科技作品竞赛**（课外学术，重学术创新与研究深度）。三类作品：
  - ① 自然科学类学术论文；
  - ② 哲学社会科学类社会调查报告和学术论文（**重一手调研数据**，臆造样本是硬伤）；
  - ③ 科技发明制作（A 类侧重技术含量、B 类侧重实用创新，**要有实物/可演示原型**）。
- **"小挑" = "挑战杯"中国大学生创业计划竞赛**（重商业可行性，提交创业计划书 + 答辩）。
- **机制**：分初评（书面）与终审（答辩/展示）两阶段；两杯交替举办、各两年一届。

## 6. 大创 / 科研训练

- **三类**：创新训练 / 创业训练 / 创业实践。
- **评审重点**：**学生主体与培养价值（过程性成长）**，不只看结果（anti_patterns E3）。
- **流程**：申报立项 → 中期检查 → 结题验收（进度安排要对齐这三阶段，每阶段有 go/no-go）。
- **预算**：分类合理、与研究内容匹配（anti_patterns D4），校级额度各校不同。
