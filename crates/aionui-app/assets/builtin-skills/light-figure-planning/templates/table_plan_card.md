# 表格规划卡 · {{table_id}}

> 一张卡 = 一张表。表也是 display item，照样要回答"它支撑哪条 claim、删了缺不缺"。
> 表 vs 图选择：**精确数值/多指标对照/无法用位置编码的离散信息** 用表；**趋势/分布/关系/比较** 优先图（表读起来慢，能图则图）。

## 上层 · 叙事（这张表为什么存在）

| 字段 | 内容 |
|------|------|
| **table_id** | `T#` 命名（与 m07 论文模板 `[表位 T1]` 占位对齐） |
| **绑定 claim** | 这张表支撑哪一条 claim（一条）；删掉它论文会缺哪一块；为什么必须是表而非图 |
| **讲什么** | 一句话：读者看完应得到什么（如"我方法在 5 数据集 3 指标上整体最优"） |
| **放哪节** | 正文哪一节 / 附录；大表（>1 页）通常进附录，正文留摘要表 |
| **优先级** | 必做 / 可做 / 可删 |

## 下层 · 规格（怎么把它排对）

### 表结构

| 字段 | 内容 |
|------|------|
| **行含义** | 每行是什么（如"每行一个方法"/"每行一个数据集"）；行数预估 |
| **列含义** | 每列是什么（如"每列一个指标/数据集"）；列数预估 |
| **表头分组** | 是否需多级表头（如指标下分 mean/std 两子列）；用 `\multicolumn`/`\multirow`（LaTeX booktabs） |
| **数值对齐** | 数字列按小数点对齐（siunitx `S` 列或 `\phantom`）；文本列左对齐 |
| **有效位数** | 各列统一有效位数/小数位（如准确率 1 位小数 76.3；别一列 76.3 一列 76.31） |
| **最优标注** | 每列/行最优值**加粗**、次优下划线；规则写清（按列还是按行比） |
| **显著性/不确定性** | 是否带 ±std / 95%CI / 显著性上标（如 *）；与正文统计口径一致（对齐 m06） |
| **单位** | 单位入表头括号，不在每个单元格重复（如"Time (s)"） |

### 排版/格式

| 字段 | 内容 |
|------|------|
| **booktabs 规则** | 用 `\toprule/\midrule/\bottomrule`，**禁竖线、禁双横线**（三线表是学术惯例）；行多时每 5 行可加 `\addlinespace` |
| **跨页处理** | 超 1 页用 `longtable`/`supertabular` 并重复表头；正文表尽量不跨页（跨页表移附录或精简列） |
| **横排处理** | 列太多放不下用 `sidewaystable`（rotating 宏包，横排整页）或 `\resizebox`/`adjustbox`（缩到 `\textwidth`，但别缩到字号过小） |
| **宽度适配** | 目标 `target_journal`+`column` 宽度（取 figure_export.py `JOURNAL_SPECS`）；表过宽优先减列/缩写表头，`\resizebox` 是最后手段 |
| **caption 位置** | 表 caption 通常在**表上方**（图在下方）；能脱离正文独立读懂；列缩写在表注（`\footnotesize` 或 `\begin{tablenotes}`）解释 |
| **source_card（必填）** | 数据来源指针：如 `m06 claim_evidence_table.md / Table 2` 或 `databases/db07-figures/...`；数值须可追溯到原始结果，禁手填臆测 |
| **target_journal / column** | 同图卡：取 `JOURNAL_SPECS` 键或 `custom`（中文刊等） |

## 投稿前自检（交卡前过一遍）

- [ ] 三线表（booktabs，无竖线）；表 caption 在上方、可独立读懂。
- [ ] 每列有效位数统一；数字按小数点对齐；单位入表头不重复。
- [ ] 最优值标注规则明确（加粗/下划线，按列还是按行）且全表一致。
- [ ] ±std/CI/显著性与 m06 统计口径一致，不夸大（不显著别标"最优"加粗误导）。
- [ ] 列缩写在表注解释；跨页/横排方案已定。
- [ ] 宽度不超目标栏宽；缩放后字号仍 ≥ 目标刊下限（别 `\resizebox` 缩成蝇头小字）。
- [ ] 这张表的数据**确实更适合表而非图**（精确数值/多指标对照）；否则改图。
