# db08 — 竞赛/项目经费预算表模板（budget_template.md）

> 服务 m17(竞赛申报，对应技能 light-competition)。本文件提供**可直接套用的经费预算表模板**与填写要点。
> ⚠ 合规底线（联动 a10 / CONVENTIONS）：
> - 预算须**与研究内容、技术路线、进度安排一一对应**，不得堆砌、不得虚报；评委会逐项追问"这笔钱买什么、为什么这个量级"。
> - 各校/各级（国家级/省级/校级）/各赛事的**开支科目名称、单项比例上限（尤其劳务费）、总额、是否允许购置设备**逐年逐校不同，凡未当场核实者一律标【待核查：以当届/本校官方预算编制说明为准】，不臆造具体比例数值。
> - 下列科目体系沿用我国财政科研项目 / 国家自然科学基金通行的"预算科目"口径（材料费、测试化验加工费、差旅费等），是大创/挑战杯校级模板最常见的分类基础；具体到某赛事请以其当届《预算编制说明》为准。

---

## 1. 适用场景与三种预算形态

| 场景 | 预算性质 | 关注点 |
|---|---|---|
| 大创 / 科研训练 / 挑战杯"大挑" | **科研经费支出预算**（学校/财政拨款怎么花） | 科目合理、与研究活动匹配、票据可报销 |
| 互联网+创业组 / 挑战杯"小挑" | **创业财务预测**（公司收入/成本/利润，3–5 年） | 收入假设可核查、成本结构真实、现金流为正的时点 |
| 互联网+创意组 | 两者之间：少量启动经费 + 简化财务预测 | 启动资金用途 + 盈利模式可行性 |

> 学术类项目用第 2 节"科研经费支出预算表"；创业类项目除启动经费外，还要用第 4 节"创业财务预测表"。两者可同时出现在一份创业计划书里（启动资金预算 + 未来财务预测）。

## 2. 科研经费支出预算表（大创/科研训练/大挑通用）

> `universal` 科目口径——下表科目体系(材料费/测试化验费/差旅/劳务等)是财政/基金通行分类,跨学科通用。复制下表，删去用不到的科目行，每行的"测算依据"必须写清"单价×数量=金额"，**不允许只写一个总数**。

| 序号 | 开支科目 | 用途说明（买什么/做什么） | 测算依据（单价×数量） | 金额(元) | 占比 |
|---|---|---|---|---|---|
| 1 | 材料费 | 实验耗材、原材料、低值易耗品、元器件 | 例：试剂 ¥X/套 × N 套 | | |
| 2 | 测试化验加工费 | 委托外单位测试、化验、检测、样品加工 | 例：测序 ¥X/样 × N 样 | | |
| 3 | 设备费 | 专用小型仪器/软件许可（多数大创**限购置，优先租用/共享**，按当届规则） | 例：传感器 ¥X × N | | |
| 4 | 差旅/会议/国际合作交流费 | 调研、采样、学术会议、参赛差旅 | 例：往返 ¥X × M 人次 | | |
| 5 | 出版/文献/知识产权事务费 | 论文版面费、查重、专利申请官费、资料费 | 例：版面费 ¥X × 1 篇 | | |
| 6 | 劳务费 | 给参与项目的校外人员/咨询专家的劳务（**学生主体一般不可领，比例常受限**【待核查本校口径】） | 例：问卷酬谢 ¥X × N 人 | | |
| 7 | 专家咨询费 | 咨询费（与劳务费分列，按本校口径） | | | |
| 8 | 其他/不可预见费 | 印刷、打印、小额杂支（通常设上限，如≤5%【待核查】） | | | |
| | **合计** | | | **=申请总额** | 100% |

**填写要点**：
- **总额对齐**：合计须等于申报书"经费申请额"，且不超过该级别（国家级/省级/校级）封顶额【封顶额待核查本校通知】。
- **匹配研究内容**：每个科目能在技术路线里找到对应活动（要做测序→才有测试费；要外出采样→才有差旅费）。无对应活动的科目删掉，别为凑数硬填。
- **留痕可报销**：高校财务多要求票据，预算时即考虑能否取得正规发票；劳务费/咨询费常需个税与名单，提前问清。
- **常见扣分点**：① 只有总额没有测算依据；② 设备费占比畸高（评委疑"买设备而非做研究"）；③ 劳务费超本校比例上限；④ 预算项与研究内容对不上；⑤ 出现与项目无关的开支（如团建、纪念品）。

## 3. 已填示例（材料化学方向大创，示意金额，仅演示填法）

> ⚠ `illustrative_only:true domain_example:材料化学` —— 下表金额为**演示性示例**,不代表任何真实项目或当届封顶额;**科目体系(材料费/测试化验费等)是 universal 通用分类照搬即可,但金额与"催化剂/XRD/SEM"等领域细节仅演示填法、非领域锁定**,非材料方向用户照搬科目结构、换自己领域的支出项,按本校通知重算。

| 序号 | 开支科目 | 用途说明 | 测算依据 | 金额(元) | 占比 |
|---|---|---|---|---|---|
| 1 | 材料费 | 催化剂前驱体、电解液、电极基底 | ¥120/套×40套 + ¥80/L×20L | 6,400 | 32% |
| 2 | 测试化验加工费 | XRD/SEM/XPS 送测、样品切割 | ¥150/样×60样 | 9,000 | 45% |
| 4 | 差旅费 | 赴合作实验室采样 2 次 | ¥800/人次×4人次 | 3,200 | 16% |
| 5 | 出版/知识产权 | 1 篇会议论文版面费 | ¥1,400×1 | 1,400 | 7% |
| | **合计** | | | **20,000** | 100% |

填法解读：测试费占比最高，因为该项目核心是表征驱动——预算结构本身就在向评委讲"我们在做实证研究"；没有设备费，因为仪器走学校共享平台（在可行性分析里已说明），避免"买设备"质疑。

## 4. 创业财务预测表（互联网+创业组 / 小挑）

> 创业类不是"花钱预算"而是"赚钱预测"。核心是**收入假设可核查 + 现金流转正时点**。所有假设进"假设登记表"，与市场分析(TAM/SAM/SOM)的 ARPU、转化率保持同一套数。

### 4.1 假设登记表（先填，让评委能逐条质疑输入）

| 假设项 | 取值 | 依据/出处 | 敏感度 |
|---|---|---|---|
| 客单价 ARPU | ¥__ | 竞品定价页/调研 | 高 |
| 付费转化率 | __% | 行业基准/试点数据 | 高 |
| 月获客成本 CAC | ¥__ | 渠道实测/类比 | 中 |
| 年留存率 | __% | 试点/行业 | 中 |
| 毛利率 | __% | 成本结构倒推 | 高 |

### 4.2 三年损益预测（简表）

| 项目 | 第1年 | 第2年 | 第3年 |
|---|---|---|---|
| 用户数(SOM 推导) | | | |
| 营业收入 = 用户×ARPU | | | |
| 营业成本(COGS) | | | |
| 毛利 = 收入−成本 | | | |
| 销售/研发/管理费用 | | | |
| 营业利润 | | | |
| 累计现金流 | | | |
| **现金流转正时点** | （标出在第几年/季） | | |

**填写要点**：
- 收入**自下而上**推（用户数×ARPU），用户数来自市场分析的 SOM，不能自上而下拍"占市场 1%"。
- 与 light-competition 的市场分析模块共用同一套假设（ARPU、转化率、CAC），跨材料数字必须一致（一致性由 a07 把关）。
- 评委必问：现金流何时转正？盈亏平衡点的用户规模？最大的成本项是什么、能否压降？CAC 与 LTV 的关系（LTV>CAC 才成立）。
- **禁区**：编造融资额/估值/用户数；市场规模自上而下虚高；毛利率脱离成本结构空写。商业数据真实性风险上报 a10。

## 5. 自审清单（提交前过一遍）

- □ 合计金额 = 申报书经费申请额，且 ≤ 本级别封顶额【核对当届通知】
- □ 每个科目都有"单价×数量"测算依据，没有光秃秃的总数
- □ 每笔预算能在研究内容/技术路线里找到对应活动
- □ 劳务费/其他费占比未超本校上限【核对本校口径】
- □ 创业类：财务预测的用户数来自 SOM，ARPU/转化率与市场分析同源
- □ 创业类：标出了现金流转正时点，假设全部进了假设登记表
- □ 无与项目无关开支，无臆造的商业数字
