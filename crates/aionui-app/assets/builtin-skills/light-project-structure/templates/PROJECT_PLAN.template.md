## No-Placeholders 规则（强制）

- 计划里**不留** `{{占位}}`、`TODO`、`见上文`、`类似地`、`等等`。每个任务自带：精确文件路径 + 完整命令/代码 + 可观察的验证结果（具体数字/状态码/文件名）。
- 任务粒度 **2–5 分钟**。超过就拆分。
- 每个任务必须可独立验证：写清"跑什么命令、期望看到什么输出"。
- 失败的尝试要记到下方「失败尝试」节，不要默默删掉——这是最有价值的信息。
