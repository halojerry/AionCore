# 高利害情形决策树（科研伦理）

> 三条最易判错、后果最重的分支，写成显式决策树。每个判定点给出"满足→走哪""不满足→走哪"。
> 这些树用于**定位下一步该找谁、按哪套流程处理**，最终裁定仍须相应权威机构/法律专业人员。

---

## 决策树 A — 疑似学术不端 → FFP 三要件 → COPE 流程

第一步：是否落入 FFP（ORI / 42 CFR Part 93）三类之一？

```
疑似不端行为
├─ 是"编造数据/结果"？ ───────────────── Fabrication
├─ 是"篡改材料/过程或选择性遗漏数据使记录失真"？ ─ Falsification
├─ 是"挪用他人想法/过程/结果/文字未署源"？ ──── Plagiarism
└─ 都不是（如方法分歧、格式瑕疵、署名礼节）→ 不按不端处理，转编辑沟通/更正
```

第二步：FFP 三要件**须同时满足**（缺一即不构成"不端"认定）：

```
落入 FFP 某类
├─ ① significant departure 显著偏离学界惯例？ ──┐
├─ ② intentionally / knowingly / recklessly 故意/明知/轻率？ ─┤ 三者皆 YES
└─ ③ preponderance of evidence 优势证据支持？ ──┘   → 可认定为研究不端
        │
        └─ 任一为 NO（典型：honest error 诚实错误 / difference of opinion 学术分歧）
              → 不认定为不端；按更正、补充数据或学术讨论处理
```

第三步：确认方向后按 COPE flowchart 选处置路径（编辑视角；作者自审时转换语境）：

```
不端类型 → 对应 COPE flowchart
├─ 抄袭（已发表）/（投稿中）   → Plagiarism flowchart（分已发表 vs 投稿中两版）
├─ 图片重复/操纵             → Image manipulation flowchart
├─ 数据造假/伪造            → Fabricated data flowchart
├─ 署名争议                → Authorship dispute / change flowchart
├─ 同行评审被系统性操纵      → Systematic manipulation of peer review
└─ 利益冲突未披露           → Undisclosed COI flowchart
        │
        └─ 通用下一步：联系作者所在机构 → 由机构诚信办(RIO)/IRB 调查
           → 期刊据结论做更正/关注声明/撤稿；保全证据，勿先公开指控
```

> 关键护栏：作者个人**不做最终"不端"裁定**，那是机构调查委员会的职责；自审阶段只做"是否需升级 + 找谁"。

**适用法域分支**：

```
研究/被调查方在哪套体系下？
├─ 美国/国际期刊语境 → 上文 ORI 42 CFR Part 93 FFP + COPE flowchart
└─ 中国语境（国内单位/国内期刊/国家科技计划项目）
    └─ 改走中国本土口径：《科研失信行为调查处理规则》失信情形清单
       + 国自然/教育部不端认定与处理口径 → 见 cn_compliance.md
       （FFP 内核相通，但失信情形更宽、管辖与处理主体不同；⚠ 条文待核）
```

## 决策树 B — IRB 审查级别：Exempt / Expedited / Full Board（45 CFR 46）

```
研究是否涉及"人类受试者"（活人 + 干预/互动 或 可识别私人信息）？
├─ 否 → 不属人体研究，无需 IRB（仍守数据/版权合规）
└─ 是
    │
    ├─ 是否落入 46.104 的"豁免类别"之一？
    │   （如：常规教育实践、匿名问卷/观察、已公开或去标识的既有数据等，
    │     且不涉脆弱人群的额外风险）
    │   ├─ 是 → 【Exempt 豁免】：仍需 IRB/授权方做"豁免认定"，作者不能自行豁免
    │   └─ 否 ↓
    │
    ├─ 风险是否"不超过最小风险(minimal risk)"且属 46.110 快速审查清单类别？
    │   （如：采血少量、无创采样、既有数据/影像、声音影像研究、问卷访谈等）
    │   ├─ 是 → 【Expedited 快速审查】：由 IRB 主席或其指定委员单独审，不必上全会
    │   └─ 否 ↓
    │
    └─ 风险大于最小风险，或涉脆弱人群（未成年/孕妇/囚犯/认知障碍）？
        └─ 是 → 【Full Board 全委员会审查】：召集会议，多数到场，多数表决
```

批准须满足 46.111 八项标准（任一不达标→打回补正）：

```
[ ] ① 风险最小化      [ ] ② 风险/受益比合理     [ ] ③ 受试者选择公平
[ ] ④ 征求知情同意    [ ] ⑤ 知情同意有文档记录   [ ] ⑥ 必要时数据安全监测
[ ] ⑦ 隐私与保密保护  [ ] ⑧ 脆弱人群额外保障
```

知情同意要素（46.116）：目的/时长/程序、风险、受益、替代方案、保密、补偿、联系方式、自愿参与且可随时退出；2018 修订要求开头放 "key information" 摘要。

> 护栏：级别认定权在 IRB，不在研究者；⚠️ 政府原文(ecfr.gov/hhs.gov)在本环境受网络策略限制，引用逐字条文须回查官方原文。中国项目按《涉及人的生命科学和医学研究伦理审查办法》等本地规定走。

**适用法域分支**：

```
涉人研究在哪套体系下审？
├─ 美国/国际语境 → 上文 45 CFR 46 三级 + 46.111 八标准
└─ 中国语境（国内医院/高校/科研机构开展的涉人生命科学与医学研究）
    └─ 走《涉及人的生命科学和医学研究伦理审查办法》三级审查路径：
       一般审查（伦理委员会会议审）/ 简易程序审查（符合低风险免会议条件）/
       免除伦理审查（限已公开或匿名既有数据等法定情形）
       → 详见 cn_compliance.md；⚠ 三级名称与适用条件以现行办法原文为准，待核
```

## 决策树 C — 引用前撤稿核查（Crossref / Retraction Watch）

端点已于 2026-06 curl 实测：单篇查询与 retraction 过滤器均返回 **HTTP 200**；命中项 `update-to[]` 字段含 `type=retraction | source=publisher | label=Retraction | updated.date-parts`。

```
拿到参考文献（含 DOI）
├─ 无 DOI → 人工核对标题/作者；高风险来源谨慎引用
└─ 有 DOI
    │  curl -s "https://api.crossref.org/works/<DOI>?mailto=you@x.org"
    │  读取 message.update-to[]
    │
    ├─ update-to[] 中有 type == "retraction"？
    │   ├─ 是 → 🛑 该文献已撤稿：删除或仅作"撤稿案例"讨论，绝不作有效证据
    │   │        撤稿原因：用该项 record-id 回查 Retraction Watch CSV
    │   │        （CSV: https://gitlab.com/crossref/retraction-watch-data）
    │   └─ 否 ↓
    │
    ├─ update-to[] 中有 type == "correction" / "expression-of-concern"？
    │   └─ 是 → ⚠️ 有更正/关注声明：核对是否影响所引结论，必要时引更正版
    │
    └─ 无 update-to 或均为正常更新 → ✅ 暂未发现撤稿，可引用
```

批量核查思路：遍历 DOI 列表 → 逐条查 Crossref → 命中 retraction 标红汇总。注意同一撤稿可能 publisher 与 retraction-watch 两个 source 重复出现，按 DOI 去重。

## 决策树 D — 涉动物研究：是否需伦理审查 + 批号来源（GB/T 35892-2018 / IACUC）

> 用户领域（奶山羊行为识别）正命中此树。核心是分清"常规生产观测"与"实验性干预"，并定位批号来源。标准号/名称实查留痕见 `../../_verification_log/R12-03-animal-ethics-standards.md`。

```
研究是否涉及"活体动物"？
├─ 否（仅已公开数据集/视频、不接触活体）→ 无需动物伦理审查
│        └─ 但仍核：所用动物数据集的采集是否已获伦理批准（用别人数据≠免责，引用时核对方声明）
└─ 是
    │
    ├─ 是否存在"实验性干预"？
    │   （取样/采血、手术、给药、标记植入(如RFID侵入式)、限饲、行为诱导、应激刺激等）
    │   ├─ 是 → 【须 IACUC 审查】实验动物福利伦理委员会审批，依 GB/T 35892-2018
    │   │        批号与涉人 IRB 同级：缺批号即 🛑 BLOCK
    │   │        审前做 3R：替代(Replacement)/减少(Reduction,联动 m05 功效分析定最小样本)/优化(Refinement)
    │   └─ 否 ↓
    │
    └─ 仅"非侵入式常规观测"？（羊场监控视频，不接触/不改变动物的日常生产管理）
        └─ 通常较低风险，但 ⚠️ 仍须机构伦理委员会**确认**是否需审查/备案
           （不可自行判定免审）；遵守现行动物福利标准——
           实验动物=GB/T 35892-2018；奶山羊等农场动物福利现主要为团标/地标
           （如 T/CAI 003-2019《农场动物福利要求 绒山羊》），引用前核标准号与现行性
```

> 护栏：审查级别与免审认定权在机构伦理委员会，不在研究者；"用视频不接触动物"不等于自动免审，须机构书面确认。农场动物福利团标/地标全文未逐条核（仅核标准号与名称），落地以机构与现行标准为准。
