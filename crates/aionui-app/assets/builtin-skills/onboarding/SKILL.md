# Onboarding

## Purpose

Get a new small business fully operational: record business context, connect tools, configure day-1 workflows, and establish a reporting baseline.

## Workflow

1. **Record business profile**
   - Industry, sub-sector
   - Team size and key roles
   - Current tools and software stack
   - Key business metrics the owner tracks (revenue, cash, customers, etc.)
   - Reporting preferences (cadence, format, recipients)

2. **Verify tool connections**
   - 财务软件：用友、金蝶、畅捷通（适合内资小微）/ QuickBooks、Xero（外资 / 跨境业务）
   - 协同与沟通：企业微信 / 钉钉 / 飞书（含审批、考勤、文档、视频会议）
   - CRM：企业微信、纷享销客、有赞CRM / HubSpot、Salesforce（外资 / B2B SaaS 场景）
   - 开票与税务：电子税务局账号、增值税开票软件、税控盘 / Ukey
   - 银行：对公账户网银（招商、工行、建行等主流）
   - 日历与会议：飞书日历 / 钉钉日历 / 企微日程
   - For each tool:
     - 已连接：核对数据通路、同步频率
     - 未连接：给出接入步骤或降级到 Excel 的临时流程
     - 缺失但对该业务关键：标记并建议优先补齐

3. **Configure day-1 workflows**
   - Invoicing: template, send schedule, payment terms
   - Expenses: categories, receipt process, reimbursement rules
   - Pipeline: stages, deal values, follow-up cadence
   - Calendar: recurring meetings, deadline reminders
   - Comms: channel setup (email, chat), notification preferences

4. **Set reporting baseline**
   - Hand off to business-reporting skill to generate initial snapshot
   - Confirm metrics that matter most to the owner
   - Set default reporting cadence (daily snapshot, Monday brief, etc.)

5. **Deliver 30-day next-steps checklist**
   - Week 1: Tool connections and day-1 workflows live
   - Week 2: First full reporting cycle, refine workflows
   - Week 3: Optimize based on initial data, add automation
   - Week 4: Review baseline, adjust reporting, plan next quarter

## Notes

- Don't assume tool choices — ask what the business currently uses
- If a business doesn't use a tool (e.g., no CRM), provide a lightweight alternative rather than pushing a tool
- The 30-day checklist should be specific to the business type and size, not generic
