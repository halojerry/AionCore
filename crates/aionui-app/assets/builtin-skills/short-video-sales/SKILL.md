# 短视频卖货脚本生成器

基于《短视频卖货》专业方法论，智能生成高转化率的短视频卖货脚本。

## 功能特性

- **7大爆款脚本模板**：产品种草式、开箱测评式、对比种草式、知识干货式、解决问题式、场景代入式、剧情植入式
- **智能模板匹配**：根据产品类型和卖点自动推荐最适合的脚本模板
- **完整脚本结构**：开场钩子 → 产品介绍 → 卖点阐述 → 使用演示 → 行动号召
- **多平台适配**：抖音、快手、视频号等平台风格自动适配

## 快速开始

### 方式一：对话直接生成

直接告诉 OpenClaw 你的产品信息和需求：

```
帮我生成一个美妆产品的短视频卖货脚本，产品是XX牌粉底液，主打持妆12小时不脱妆
```

### 方式二：使用脚本工具生成

```bash
python3 workspace/skills/short-video-sales/scripts/generate_script.py \
  --product "XX牌粉底液" \
  --category "美妆" \
  --selling-point "持妆12小时不脱妆" \
  --template "product-seeding" \
  --duration 30
```

## 脚本模板说明

### 1. 产品种草式 (product-seeding)
**适用场景**：日常用品、美妆护肤、食品饮料
**核心逻辑**：生活场景切入 → 引出产品 → 产品优势 → 产品效果
**最佳时长**：30-45秒

### 2. 开箱测评式 (unboxing-review)
**适用场景**：数码产品、新品首发、高端商品
**核心逻辑**：开箱氛围营造 → 产品展示 → 功能测试 → 真实体验
**最佳时长**：45-60秒

### 3. 对比种草式 (comparison)
**适用场景**：有竞品对比优势的产品
**核心逻辑**：常规方案弊端 → 竞品对比 → 产品优势突出 → 效果验证
**最佳时长**：30-45秒

### 4. 知识干货式 (knowledge)
**适用场景**：需要专业知识的产品（母婴、健康、教育）
**核心逻辑**：话题引入 → 干货知识 → 产品特色 → 价值强调
**最佳时长**：45-60秒

### 5. 解决问题式 (problem-solving)
**适用场景**：解决特定痛点的产品
**核心逻辑**：痛点问题抛出 → 解决方案 → 产品演示 → 效果证明
**最佳时长**：30-45秒

### 6. 场景代入式 (scenario)
**适用场景**：情感类产品、礼品类、生活方式类
**核心逻辑**：场景营造 → 情感共鸣 → 产品融入 → 行动号召
**最佳时长**：30-60秒

### 7. 剧情植入式 (story)
**适用场景**：有故事性的产品、品牌宣传
**核心逻辑**：故事开头 → 冲突/转折 → 产品解决 → 升华主题
**最佳时长**：60-90秒

## 命令行参数

```bash
python3 scripts/generate_script.py [选项]

选项:
  --product TEXT        产品名称（必填）
  --category TEXT       产品类别，如：美妆/数码/食品/母婴等
  --selling-point TEXT  核心卖点，多个用逗号分隔
  --price TEXT          价格信息
  --target TEXT         目标人群
  --template TEXT       脚本模板类型
  --duration INTEGER    目标视频时长（秒）
  --tone TEXT           语气风格：professional/casual/humorous/emotional
  --output TEXT         输出文件路径
```

## 模板匹配建议

| 产品类型 | 推荐模板 | 原因 |
|---------|---------|------|
| 美妆护肤 | product-seeding | 真实使用体验展示效果最好 |
| 数码3C | unboxing-review | 开箱仪式感强，功能展示直观 |
| 食品饮料 | product-seeding | 视觉诱惑力强，种草效果好 |
| 母婴用品 | knowledge | 需要专业知识建立信任 |
| 家居清洁 | problem-solving | 痛点明确，效果对比明显 |
| 服装配饰 | scenario | 穿搭场景代入感强 |
| 教育培训 | knowledge | 干货价值高，转化率高 |
| 健康保健 | problem-solving | 解决健康问题，需求刚性 |
