# Intent Alignment Card 模板

Intent Alignment Card 是一次 AI 协作前的“对齐卡”。

它解决的问题：用户以为自己在问 A，其实真正想解决的是 B。先把 B 说清楚，Agent 才不会在 A 上跑偏。

## 模板

```markdown
# Intent Alignment Card

## 真实意图
{我真正要解决什么}

## 表层任务
{我原本打算让 AI 做什么}

## 刺点
{当前最关键的矛盾 / 单点故障 / 反常识点}

## 关键差异
{表层任务和真实意图哪里不一样}

## 触发时机
{我什么时候会用它？什么时候最需要但不会用？}

## 不做什么
{本次明确不解决的东西}

## 成功信号
{什么结果说明这次协作有用}

## 跑偏信号
{出现什么就说明 Agent 跑偏了}

## 必须由人拍板的判断
{价值取舍、风险承担、最终决策}

## 可以交给 Agent 的部分
{生成、整理、搜索、比较、草拟、检查}
```

## 判断标准

好的对齐卡，读完能回答四句话：

1. 我到底要解决什么？
2. 这件事最容易在哪个地方自欺？
3. Agent 该做什么、不该做什么？
4. 哪些判断我不能外包？
