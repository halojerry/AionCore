# 定性分析框架：成长性判断与核心亮点

> 本文件按需加载。SKILL.md 主流程在需要深入定性分析时引用。

定性分析的核心任务是回答两个问题：**这家公司在成长吗？为什么值得关注？**

## 4.1 先做初步分型（定性与定量的共同透镜）

进入五维度诊断之前，先用增速、股息率、行业地位、主线归属做一次**初步分型**（成长股 / 成熟龙头 / 红利价值；三型的完整定义与对应估值模式见 `references/quantitative_framework.md` §5.1）。分型不是定量章节专属——它同时决定**本节定性分析的重心**：对不同类型，五维度里哪些是 first-class、哪些近乎无关，差异很大。分型是基于初步证据的**可推翻假设**，可在分析过程中回头修正（尤其类型切换期：成长股增速降档转成熟、成熟龙头被新主线重新点燃）。

| 类型 | 定性分析重心（本节哪些维度优先 + 额外要看什么） | 对应估值模式 |
|------|--------------------------------|------------|
| **成长股** | 重心在**维度一（收入增速质量：量 vs 价、公司 alpha vs 行业 beta）**与**维度四（业务结构与壁垒：远期空间 TAM、第二曲线、渗透率提升）**。额外看：赛道天花板、产品管线/产能投放节奏、主线持续性。**判断逻辑：增速能否兑现 > 当期利润高低**；自由现金流为负在重投入期可容忍。 | 5.3 隐含预期 + 5.4 三情景；PE band 仅作下行参考 |
| **成熟龙头** | 重心在**维度二（利润含金量与利润率稳定性）**、**维度五（ROE 可持续性与杜邦结构）**、**维度三（现金流）**。额外看：增速降档后是否仍有定价权与份额、护城河是否在维持、分红 + 回购等股东回报力度。**判断逻辑：看"稳"不看"快"**，增速从高速回落到中速本身不是利空，盈利能力松动才是。 | 5.2 PE/PB 历史分位带为核心 |
| **红利价值** | 重心在**维度三（现金流：自由现金流能否覆盖股息）**、**维度四（业务稳定性与永续经营属性）**、资产负债安全。维度一增速基本不作主线（低速可接受），维度五 ROE 看稳定性而非高低。额外看：派息率与连续派息历史、资本开支是否轻、行业是否夕阳。**判断逻辑：稳定与永续 > 成长**。 | 5.2 股息率分位 + 股息率/利差、DDM |

下面的五维度模型是**通用清单**，但要按上表的重心**分配权重**，不要对三类公司机械地等权打分。

## 4.2 成长性诊断：五维度模型

按照以下五个维度逐一评估，每个维度给出"强/中/弱/恶化"判断（权重按 4.1 分型调整）：

**维度一：收入增长的质量**
- 观察指标：近 4-8 个报告期的营业收入同比增速序列。
- 判断标准：
  - **强**：连续 3 期以上正增长且增速稳定或加速。
  - **中**：正增长但增速波动较大，或刚从负增长恢复。
  - **弱**：增速持续下滑或在低个位数徘徊。
  - **恶化**：出现负增长，尤其是连续 2 期以上。
- 深层追问：增长是来自量（市占率扩张、新客户）还是价（提价能力）？是行业 beta 还是公司 alpha？用 `main-business-product` 判断哪些产品线在驱动增长。

**维度二：利润增长的含金量**
- 观察指标：归母净利润增速、扣非净利润增速、毛利率和净利率的趋势。
- 判断标准：
  - 利润增速 ≥ 收入增速 → 经营杠杆释放或效率改善，含金量高。
  - 利润增速 < 收入增速 → 增收不增利，需追查成本端原因。
  - 扣非利润与归母利润严重背离 → 利润质量存疑，关注非经常性损益占比。
- 关键信号：毛利率连续 2 期以上下滑是成长性减弱的早期预警。

**维度三：现金流验证**
- 观察指标：经营活动现金流净额 vs 净利润；自由现金流（经营现金流 - 资本支出）。
- 判断标准：
  - 经营现金流/净利润 > 0.8（长期均值） → 利润可回收，成长有实质支撑。
  - 经营现金流/净利润持续 < 0.5 → 利润可能被应收/存货吞噬，成长虚实存疑。
  - 自由现金流持续为负但收入高增 → 可能处于重投入扩张期，需判断资本开支回报周期。
- 注意：对于重资产行业（基建、制造）和高速扩张期企业，短期自由现金流为负不一定是负面信号，需要结合 ROIC 和资本开支去向判断。

**维度四：业务结构与竞争壁垒**
- 数据来源：`company`（行业、简介）、`main-business-product`（产品结构）、`main-business-region`（地区结构）、联网检索（行业格局）。
- **脚本已做确定性计算**：`pack` 产出的 `business_structure.product_analysis` 和 `business_structure.region_analysis` 包含跨期对比——产品/地区的收入增速、占比变化、毛利率变化。模型直接引用这些计算结果做判断，不用自己算。
- 判断要素：
  - **收入集中度**：单一产品/客户占比 > 50% 是集中风险，但也可能是细分领域龙头的表现。`product_analysis.rankings.share_expanding/shrinking` 直接显示结构变化方向。
  - **业务多元性**：多产品线且新产品贡献上升 → 第二增长曲线正在形成。看 `product_analysis.product_trends` 中哪些产品增速远高于整体、占比在提升。
  - **出海/全球化**：`region_analysis.globalization_summary` 给出海外收入占比变化趋势、海外 vs 国内增速差、毛利率差。海外增速显著高于国内 + 海外毛利率不低于国内 = 出海逻辑成立；反之需警惕汇率侵蚀或低价竞争。
  - **竞争壁垒类型**：规模效应、品牌溢价、技术专利、客户切换成本、网络效应、牌照/准入。
  - **行业位置**：在行业中处于领导者/挑战者/跟随者的哪个位置？
- 注意：竞争壁垒和行业地位等外部判断，如果无法仅从财务数据得出，必须联网检索并引用来源。不要编造。

**维度五：再投资能力与ROE分解**
- 观察指标：ROE 序列、杜邦三因子拆解（净利率 × 总资产周转率 × 权益乘数）。
- 判断标准：
  - ROE > 15% 且稳定 → 优质企业特征。
  - ROE 上升由净利率驱动 → 最健康的路径。
  - ROE 上升由杠杆驱动（权益乘数上升） → 需关注债务风险。
  - ROE 下行但收入仍增 → 可能因大额资本开支稀释，需判断是否为合理扩张。
- 与成长的关系：可持续增长率 ≈ ROE × 留存收益比率。ROE 高且稳定的公司，内生成长能力更强。

## 4.3 成长阶段定位

综合五个维度的证据，将公司归入以下成长阶段之一。**成长阶段与 4.1 分型应当自洽**：高速/稳健成长期通常对应成长股，成熟稳定期对应成熟龙头或红利价值，下行/困境期则要重新审视分型是否成立。两者若矛盾（如判成长股却落在成熟稳定期），说明分型需要修正。

| 阶段 | 特征 | 典型财务画像 |
|------|------|------------|
| **高速成长期** | 收入增速 > 20%，利润增速更快，市场份额快速扩张 | 高收入增速、毛利率稳定或上升、自由现金流可能为负（重投入） |
| **稳健成长期** | 收入增速 10-20%，盈利能力稳定提升，竞争格局趋于稳定 | 收入利润同步增长、ROE 稳中有升、现金流改善 |
| **成熟稳定期** | 收入增速 < 10%，利润率稳定，强现金流生成能力 | 增速放缓但利润率高、自由现金流充沛、高分红/回购 |
| **增速放缓/拐点待察期** | 增速连续下台阶，盈利能力出现压力但尚未恶化 | 收入增速下滑、毛利率松动、ROE 下行但仍在合理区间 |
| **下行/困境期** | 收入负增长，利润大幅下滑或亏损，资产负债表承压 | 负增长、利润率大幅萎缩、现金流恶化、杠杆上升 |

## 4.4 核心亮点提炼："为什么要关注这只股票"

在成长诊断和阶段定位后，提炼 2-4 条核心亮点。**亮点要落在该类型的核心矛盾上**：成长股的亮点是成长动力与远期空间，成熟龙头的亮点是盈利稳定性与股东回报，红利价值的亮点是高且可持续的现金分红与永续属性。每条亮点必须有数据支撑：

- **亮点的来源**：成长动力（新产品/新市场/产能释放）、效率提升（利润率改善/费用率下降）、壁垒强化（市占率提升/客户黏性增加）、资本配置优化（资本开支收缩后现金流释放）、行业景气上行。
- **亮点的检验**：每条亮点都要问"反方怎么看？"如果反方观点同样有合理依据，将其写在亮点后面作为对冲。
- **措辞边界**：亮点是"值得关注的理由"，不是"买入推荐"。
