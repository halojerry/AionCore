# 定量分析框架：估值阶段与隐含预期

> 本文件按需加载。SKILL.md 主流程在需要深入估值分析时引用。

定量分析的核心任务是回答：**当前价格隐含了什么样的未来预期？这个预期合理吗？**

**但在回答之前，必须先做两步前置判断**：(1) 这家公司处于哪个**生命周期**——决定"看远期还是看当期"；(2) 这家公司属于哪个**行业 / 商业模式**——决定"用哪个倍数 / 哪个指标作为主估值锚"。两轴正交，必须同时做。A 股最常见的估值错误不是算错数，而是**用错估值模式**——拿 PE band 去框一只逻辑还没兑现的成长股，或拿通用 PE 去给一家重资产周期股讲故事，或拿远期空间给一只永续红利股讲故事。先分型（5.1），再选模式（5.1 + `references/industry_valuation_library.md`），然后才进入隐含预期（5.3）和阶段定位（5.6）。

## 5.1 双轴分型与估值模式选择（估值的前置判断）

估值模式由两条正交的分型轴共同决定，**两轴都要做**：

### 5.1.A 轴 1：生命周期分型（看远期还是看当期）

先把公司归入以下三型之一（可以是「主导型 + 次要属性」，但必须有一个主导判断）。分型依据来自第四节的成长诊断、`daily-basic` 的股息率与增速、以及 5.5 节的主线归属。**分型在 4.1 已先行做出，并决定了第四节定性分析的重心**；本节是同一分型在定量侧的延续——下表给出每型对应的看重维度。

| 类型 | 识别特征 | 看重什么 | 历史 band 的作用 |
|------|---------|---------|------------------|
| **成长股** | 处在当前主线/风口（5.5 强连接），收入高增、有未被证实的远期预期，估值随叙事大幅跳变 | 未来增速 + 远期期权（行业空间 TAM × 渗透率 × 公司市占）+ 主线持续性 | **弱**：历史样本短且估值随预期跳变，分位带只作「逻辑被证伪时会跌到哪」的下行参考，不作贵贱锚 |
| **成熟龙头** | 行业龙头但增速已降档、盈利可预测、竞争格局稳定（典型如宁德时代） | 盈利稳定性 + 增速台阶（从高增到中速）+ 股东回报（分红 + 回购占净利润比例） | **核心工具**：判断当前估值处于公司自身历史的便宜/中枢/贵区间 |
| **红利价值** | 业务稳定、现金流强、永续经营属性强、高派息、低增速 | 股息率水平 + 派息可持续性（派息率、自由现金流覆盖）+ 永续属性 + 资产负债安全 | 看 **`dv_ttm`（股息率）分位** 为主，PE/PB band 为辅——红利股贵贱由股息率而非 PE 决定 |

**生命周期型本身的边界**：分型是**可被推翻的假设**，要标注置信度。处于类型切换期的公司（成长股增速降档转成熟、成熟龙头被新主线重新点燃）最难判断，必须说明。类型决定「历史中枢是否仍然有效」：增速降档的公司，过去高增长期的中枢已失效，band 要相应缩短窗口或明确弃用。

### 5.1.B 轴 2：行业 / 商业模式分型（用哪个倍数作为主估值锚）

**不要默认 PE**。先按以下决策树定主估值指标，再查 `references/industry_valuation_library.md` 的行业方法库做精化：

1. **盈利是否稳定且可预测？** 是 → PE / PEG 可作为主指标之一（仍要受第 3 问修正）；否（亏损 / 微利 / 利润大幅波动）→ 跳过 PE，进入第 2 问。
2. **收入是否高增但利润尚未释放？** 是 → 主用 **PS / EV-Sales**；增长质量好 + 行业空间清晰时叠加远期 PE。
3. **资产结构是否重 / 是否强周期？** 是（折旧高、capex 占收入比高、产能利用率显著影响利润、价格随周期大幅波动）→ 主用 **PB / EV-EBITDA / 中周期利润 PE**，**不能用景气顶利润算 PE**。
4. **是否多业务并行、且各业务估值逻辑差异显著？** 是 → **SOTP 分部估值**，对每段业务套合适方法分别估值后求和。

特殊商业模式补丁：
- **订阅制 / SaaS**：主用 **EV/ARR + PS + Rule of 40**；ARR、净留存率（NDR）、毛利率三者缺一不可。
- **创新药 / 临床阶段**：主用 **rNPV / DCF**；按管线成功率与峰值销售推算。
- **金融 / 银行 / 保险**：主用 **PB + P/EV / P/NBV + ROE-PB 联动**；不要单看 PE。
- **地产 / 基建**：主用 **PB + NAV**；表内土储与合同负债比当期利润更说明问题。
- **半导体 / AI 算力 / 机器人 / 光模块 / CXO / 周期资源品 / 军工** 等典型 A 股赛道：查 `references/industry_valuation_library.md` §2 行业方法库表。

### 5.1.C 双轴合成示例

| 双轴组合 | 主估值锚 | 历史 band 作用 |
|---|---|---|
| 半导体设备 + 成长股 | **PS 历史分位 + 同业 PS 中位**；远期 PE 兜底 | PE band 仅作下行参考 |
| 晶圆制造 + 成熟龙头 | **PB / EV-EBITDA + 中周期 PE** | PB band 为主 |
| 银行 + 红利价值 | **PB + 股息率 + P/EV 联动** | 股息率 band 与 10Y 国债利差并用 |
| 消费品龙头 + 成熟龙头 | **PE 历史分位 + 同业 PE 中位 + 股东回报率** | PE band 为核心工具 |
| IC 设计 + 成长股 | **PEG / PS + 远期 PE** | PE band 失真，看产品周期与库存周期 |
| 创新药 + 成长股 | **rNPV / DCF 情景估值** | PE / PS 失效，看管线推进 |
| 平台型互联网 | **SOTP**：成熟业务 PE / FCF + 新业务 PS | 整体 PE 容易误导 |

**两轴合成后再回到 5.3**：选定主估值锚之后，5.3 的"隐含预期推算"也要切换成该指标——例如主锚是 PS，则推算的是"市场隐含的远期收入增速"而非"隐含 EPS 增速"。

## 5.2 估值快照（按主估值锚组织，不默认 PE）

**第一步：先按 5.1 双轴定出主估值锚 + 辅助锚**，再从 `daily-basic` 取数。不要每次都先报 PE——主锚不是 PE 时，PE 在快照里只作"辅助参考"。

`daily-basic` 提供的估值字段：

| 指标 | 数据来源 | 何时作为主锚 |
|------|---------|------------|
| PE（TTM） | `pe_ttm` | 盈利稳定 + 行业 cheat sheet 首选 PE 时（消费、CXO、互联网平台、IC 设计成熟期等） |
| PB | `pb` | 重资产 / 周期 / 金融 / 地产；高 ROE 公司合理 PB 更高 |
| PS（TTM） | `ps_ttm` | 高增长但利润未释放（半导体设备、机器人、工业软件、SaaS、AI 应用） |
| 股息率（TTM） | `dv_ttm` | 红利价值型主锚；高分位 = 相对自身历史更便宜 |
| 总市值 | `total_mv` | 判断公司规模和流动性级别，所有类型都看 |

**daily-basic 字段覆盖不到的主锚**（必须靠其他数据组合得到，或联网取）：
- **EV/EBITDA**：市值 + 有息负债 - 现金 ≈ EV；EBITDA 从 `income` 表算（净利润 + 利息 + 税 + 折旧摊销，折摊看 `cashflow` 表）。
- **EV/Sales、EV/ARR**：EV ÷ TTM 收入；SaaS 的 ARR 需要从公告或路演材料取，本 skill 无现成接口。
- **rNPV、DCF、NAV、SOTP**：完全靠定性 + 行业研究构建，本节只给"推算思路"，不给"快照倍数"。

**第二步：取历史分位带**。Tushare 没有现成的估值分位接口，用 `valuation-band` 数据集（脚本从多年 `daily_basic` 历史确定性计算）得到 `pe_ttm`、`pb`、`ps_ttm`、`dv_ttm` 在近 1/3/5 年的 min/p25/median/mean/p75/max 和**当前分位**。读法按 5.1 双轴区分：

- **PE 为主锚**（消费龙头、CXO、IC 设计成熟期等）：看 `pe_ttm` 分位。
- **PB 为主锚**（晶圆制造、银行、地产、周期资源品）：看 `pb` 分位；PE 分位若与 PB 分位严重背离，说明利润正在被周期/折旧扭曲，应以 PB 为准。
- **PS 为主锚**（半导体设备、SaaS、机器人）：看 `ps_ttm` 分位；PE 分位失真不必勉强解读。
- **股息率为主锚**（红利价值）：看 `dv_ttm` 分位——高分位代表相对自身历史更便宜（股息率更厚）。
- **成长股共同陷阱**：分位带只作下行参考，不作贵贱锚。
- **共同陷阱**：上市/高增长时间短则样本不足、代表性弱；增速明显降档时，过去高增长期形成的中枢已失效，应缩短窗口或弃用历史分位，并在结论中说明。

```bash
python scripts/data_fetcher.py fetch valuation-band 600519.SH --years 5
```

**第三步：取同业可比中位数**（5.1 双轴选出的主锚维度上）。脚本无自动同业接口，需要构造 3–5 家可比清单后逐家 `fetch daily-basic --limit 1` 自算中位。流程见 `references/industry_valuation_library.md` §4。**至少要给出同业中位**，否则 5.6 的估值结论置信度直接降一档。

## 5.3 逆向推算隐含预期（按 5.1 主锚分路径，不强制 PE）

通过当前市值反推**主估值锚维度**的市场隐含预期，再与公司实际能力对比。**主锚不是 PE 时，禁止把推算路径塞回 PE 框架**——例如 PS 主锚的票应反推"市场隐含的远期收入与利润率正常化路径"，把它当成"扣非为负所以无法逆推 PE = 不能做隐含预期"是错的。

**步骤 A：锚定主锚的基线**

| 主锚 | 基线 | 数据来源 |
|------|------|---------|
| PE / PEG | 最近 4 个季度归母净利润（TTM）；非经常项目高时同步参考扣非 | `income` + `financial` |
| PS / EV-Sales | 最近 4 个季度营业收入（TTM）；同步看毛利率与净利率正常化空间 | `income` |
| EV/EBITDA | TTM EBITDA = 净利 + 利息 + 税 + 折摊 | `income` + `cashflow`（折摊） |
| EV/ARR | 最新 ARR、NDR、毛利率（公告 / 路演披露） | 公告原文 |
| PB | 最新一期 BVPS + 中周期 ROE（避免景气顶 ROE） | `balance` + `financial` |
| 股息率 / DDM | 最近 1 年每股股利 + 派息率 + 10Y 国债利率 | `rewards` + 联网 |

**步骤 B：按主锚反推隐含变量**

| 主锚 | 反推什么 | 关键启发式（粗略，必须用同业 / 历史校准） |
|------|---------|--------------------------------------|
| **PE / PEG** | 隐含 EPS 增速 | PE<15 ≈ 隐含 0–低增速；PE 15–25 ≈ 10–20%；PE 25–40 ≈ 20–35%；PE>40 ≈ 极高增速或叙事溢价。**ROE 锚定**：高 ROE（>20%）支撑更高 PE；低 ROE 同 PE 隐含更激进。利率下行 / 行业景气上行可上修。PEG > 2 = 估值对增长过度乐观。 |
| **PS / EV-Sales** | **隐含的远期收入 + 利润率正常化路径** | 反推法：设远期 N 年合理 PE 区间（依同业成熟态）、远期合理净利率 → 远期净利 = 远期市值 ÷ 远期 PE → 远期收入 = 远期净利 ÷ 远期净利率 → 隐含收入 CAGR。**关键问题**："市场是否在定价利润率从 X% 正常化到 Y%"——若需要的正常化幅度公司无证据可达，则隐含预期偏激进。 |
| **EV/EBITDA** | 隐含的远期 EBITDA 与 margin 改善 | 与 PS 类似，多了"EBITDA margin 在 N 年内由 X 提升到 Y"的隐含假设。 |
| **EV/ARR** | 隐含的 ARR 增速 + NDR 持续性 | EV/ARR 6× ≈ 中性增长（20–30%）+ NDR ≥ 110%；> 10× ≈ 高增长（40%+）+ 高 NDR；< 4× ≈ 增长放缓 / 留存恶化定价。 |
| **PB** | 隐含的中周期 ROE | PB ≈ (ROE - g) / (r - g)。当前 PB 高于历史中位，意味着市场在定价"中周期 ROE 高于历史"。**周期股关键**：景气顶 ROE 不能用，必须用中周期。 |
| **股息率 / DDM** | 隐含的派息可持续性 | 当前股息率显著低于 10Y 国债利率 + 风险溢价 → 市场在定价"未来派息会增长"；显著高 → 市场在定价"派息会萎缩"。 |
| **rNPV / SOTP** | 隐含的临床成功率 / 各分部估值权重 | rNPV：反推市场对管线成功率的隐含赋值；SOTP：反推市场对哪一段业务的估值溢价 / 折价。 |

**注意（PE 之外的主锚共同陷阱）**：本 skill 没有一致预期数据，所有"远期 N 年增速、利润率、ROE"都是你基于历史趋势与业绩预告的假设，必须显式标注为假设。

**步骤 C：与实际成长能力对比**
- 将步骤 B 推算的隐含变量，与第四节定性判断的实际能力对照：
  - **隐含 < 实际能力** → 市场可能低估了对应维度（收入空间 / margin 正常化 / ROE 持续性 / 派息可持续性 / 管线价值）。
  - **隐含 ≈ 实际能力** → 定价合理，上行空间取决于超预期。
  - **隐含 > 实际能力** → 市场过于乐观，或依赖尚未兑现的叙事（需要催化剂验证）。

**重要**：5.3 节得出的隐含预期判断必须用 5.5 节的主线归属做最终修正，否则容易系统性误判 A 股估值。

## 5.4 三情景估值框架（沿用 5.1 选定的主估值锚，不强制 PE）

**关键纪律：三情景的"推算变量"必须与 5.1 选定的主估值锚一致。** 如果 5.1 选的主锚是 PS，三情景就推"远期收入 × 远期合理 PS"，而不是再回到"远期净利 × 远期 PE"——后者是 5.1 已经主动放弃的指标，重新带回会直接抵消分型努力。常见错误：5.1 选了 PS 但 5.4 三情景仍按 PE 推算，结果 PS 只能在表底以"若改用 PS"作为脚注出现。

在隐含预期分析的基础上，构建三种情景：

**基准情景（Consensus）**
- 假设：当前主营驱动延续，收入 / 资产 / 派息按近 2 年趋势延展，主锚维度的合理倍数维持自身历史中位 + 同业中位的均值附近。
- 推算方式：按下表选模板。
- 与当前市值比较得出隐含回报。

**乐观情景（Bull）**
- 假设：核心驱动加速（新产品放量 / 行业景气上行 / 市占率提升 / 利润率正常化 / 净息差扩张 / 派息率提升等，按行业不同），主锚倍数向上分位扩张。
- 说明什么条件触发这个情景，以及需要什么证据来验证。

**压力情景（Bear）**
- 假设：核心驱动放缓或反转（需求萎缩 / 竞争加剧 / 政策冲击 / 减值 / 派息缩水），主锚倍数向下分位收缩。
- 说明什么事件会导致这个情景。

**给三情景配概率权重（成长股 / B 型期权必做）**：三情景只列"如果……值多少"还不够，要给每个情景一个**主观概率**，才能算出期望值、才知道当前价格押在了哪个情景上。概率的主要依据是**成长成功率档位**（见 `references/growth_success_rate.md`）——它估计的是"管理层能把成长 / 期权兑现出来的概率 P(execution)"，正是该往乐观还是悲观倾斜概率质量的判断。映射起点见 §5.4.C；概率是显式标注的主观假设，不是精确模型。成熟龙头 / 红利价值可省略概率加权，回到单点基准 + 上下区间。

### 5.4.A 按主估值锚分模板

按 5.1 选定的主锚选下表对应模板。**模板的"推算变量"和"合理倍数"两列才是关键，不能混用**。

| 主锚 | 推算变量（远期 N 年的） | 合理倍数 | 情景市值 |
|------|--------------------|---------|---------|
| **PE / PEG** | 远期净利润 = 当前 TTM 净利 × (1+EPS 增速)^N | 远期合理 PE（同业中位 + 自身历史 + 增速降档后修正） | 净利 × PE |
| **PS / EV-Sales** | 远期收入 = 当前 TTM 收入 × (1+收入增速)^N | 远期合理 PS（同业中位 + 自身历史 + 利润率正常化能力修正） | 收入 × PS（EV-Sales 时再调整为市值） |
| **EV/EBITDA** | 远期 EBITDA = 远期收入 × EBITDA margin | 远期合理 EV/EBITDA | EBITDA × 倍数 - 净有息负债 |
| **EV/ARR**（SaaS） | 远期 ARR = 当前 ARR × (1+ARR 增速)^N，叠加 NDR | 远期 EV/ARR（与 NDR、毛利率、Rule of 40 联动） | ARR × 倍数 - 净有息负债 |
| **PB / EV-EBITDA（重资产 / 周期）** | 远期净资产 = 当前 BVPS × (1 + (1-派息率)×ROE)^N；或资源品按吨盈利 × 产能 | 远期合理 PB（结合中周期 ROE 而非景气顶 ROE） | 净资产 × PB |
| **股息率 / DDM**（红利价值） | 远期股息 = 当前每股股利 × (1+股息增速)^N | 目标股息率（10Y 国债利率 + 风险溢价） | 股息 ÷ 目标股息率 |
| **rNPV / DCF**（创新药 / 公用） | 各管线 / 现金流的风险调整折现 | 折现率 r、永续增速 g、成功率 p | NPV 直接相加 |
| **SOTP**（多业务） | 各业务分别用对应模板 | 各业务对应合理倍数 | 求和 - 中心成本 |

### 5.4.B 输出格式（按主锚分别取表头）

| 主锚类型 | 表头 |
|---------|------|
| PE 主锚 | `情景 \| 核心假设 \| 远期净利 \| 合理PE区间 \| 隐含市值 \| 与当前偏离` |
| PS 主锚 | `情景 \| 核心假设 \| 远期收入 \| 远期净利率（说明） \| 合理PS区间 \| 隐含市值 \| 与当前偏离` |
| PB 主锚 | `情景 \| 核心假设 \| 远期BVPS \| 中周期ROE \| 合理PB区间 \| 隐含市值 \| 与当前偏离` |
| EV/EBITDA 主锚 | `情景 \| 核心假设 \| 远期EBITDA \| 合理EV/EBITDA \| 隐含EV \| 隐含市值 \| 与当前偏离` |
| 股息率主锚 | `情景 \| 核心假设 \| 远期每股股利 \| 目标股息率 \| 隐含市值 \| 与当前偏离` |
| rNPV / SOTP | 直接列各部分 NPV / 各业务估值 + 求和 + 与当前偏离 |

**辅助锚可在表后补一句对照**（如 "对照口径：若 2028 年净利率达 15%、给予 PE 30×，对应市值 X"），但不能反过来用辅助锚作为主推算路径——否则就违背了 5.1 的分型结论。

### 5.4.C 概率加权期望市值（接成长成功率档位）

成长股 / 已确认 B 型期权的票，在 5.4.B 三情景表后再加一行**期望市值**：

```
期望市值 = P(乐观)×乐观市值 + P(基准)×基准市值 + P(压力)×压力市值
```

三档概率由**成长成功率档位**给出起点（档位定义见 `references/growth_success_rate.md §3`、映射规则见 §4，模型可按个案微调并说明依据）：

| 成长成功率档位 | 乐观 / 基准 / 压力 | 直觉 |
|---|---|---|
| 高成功率 / 可信赖 | 40 / 45 / 15 | 概率质量推向乐观，相信团队能兑现上限 |
| 中性 / 混合 | 25 / 50 / 25 | 围绕基准、对称分布 |
| 低成功率 / 存疑 | 15 / 40 / 45 | 概率质量推向压力，新方向当期权而非基准 |

**B 型期权的计入也由档位决定**：高 → 期权可进乐观情景核心驱动、甚至部分进基准；中 → 仅进乐观情景；低 → 不进主路径，只在表后做"若成功则……"对照脚注。

把期望市值与当前市值比，得出"概率加权后的隐含回报"。这一步把"成功率"真正量化进估值，而不是停在定性形容。注意：概率是主观假设，必须显式标注，并与 5.5 主线归属、第一部分财务质量交叉印证——三者矛盾时降置信度。

## 5.5 主线归属判断（A股估值的关键调节变量）

**核心哲学：A股是高度叙事驱动的市场。一只股票如果处于当前市场主线内，估值往往会被资金提前给出——市场会把未来 1-3 年甚至更远的预期提前打入当前价格。如果脱离主线，估值就必须靠业绩实打实兑现才能扩张。这个判断不是锦上添花，而是定量分析中无法回避的关键修正项。**

忽略主线归属做估值判断，是 A 股研究最常见的系统性错误：要么把主线内合理的高估值误判为泡沫，要么对主线外的低估值机械地认为是机会。

### 5.5.1 判断公司是否处于市场主线内

**第一步：识别当前市场主线是什么**

- 主线是阶段性的，必须结合**当前时点**判断。不同阶段主线不同——可能是 AI/算力，可能是高股息红利，可能是新能源/出海/创新药/国产替代/军工/低空经济等等。
- 主线的可观察特征：
  - 成交活跃度集中在该方向相关板块。
  - 机构调研频率上升、卖方研报密集出现。
  - 政策和产业催化频繁出现。
  - 媒体和投资者讨论高度集中。
- **绝对不要凭训练时的记忆判断当前主线**——主线随时间快速变化，必须通过近期市场观察和联网检索得出。如果检索不到清晰证据，明确说明"主线判断置信度低"。

**第二步：评估公司与主线的连接强度**

| 连接强度 | 特征 | 估值表现 |
|---------|------|---------|
| **强连接（核心标的）** | 主营业务直接受益于主线，是市场公认的代表性公司 | 估值显著膨胀，PE 50+、PEG 2+ 都可能被市场接受 |
| **弱连接（边缘受益）** | 业务有部分关联但非主导，配套或间接受益 | 阶段性溢价，主线降温即回落 |
| **无连接（脱离主线）** | 业务与当前主线无关，或在已经退潮的旧主线中 | 估值锚回到基本面本身，甚至可能因被资金抛弃而折价 |

**第三步：用资金和数据验证连接强度**

- 公募基金/北向资金持仓近期是否上升 → 主流资金是否在认可。**数据边界**：交易所自 2024 年起取消了个股层面北向持股的高频披露，只剩季度数据；公募持仓也仅季度披露。不要假装能拿到实时北向数据，更不要编造——拿不到时改用季度持仓 + 联网检索卖方观点。
- 卖方研报频率近期是否明显上升 → 市场共识是否在形成（需联网检索，本 skill 无研报数据）。
- 个股近期涨跌幅是否显著跑赢/跑输行业 → 资金偏好的差异（用 `daily` 自算个股区间涨跌，行业对比需联网或同业取数）。
- `institutional-research` 中调研频率、参与机构类型、问题关键词是否集中在当前主线 → 市场关注度和产业问题焦点。
- `announcements` 中是否有主线相关订单、合作、扩产、定增、股权激励等公司级催化 → 主线连接是否有公司披露支撑。
- 上述信号有助于校验你对主线归属的判断，避免凭主观印象误判。

### 5.5.2 主线归属对估值判断的修正规则

把 5.3 节的隐含预期结论用以下规则修正：

| 主线归属 | 估值容忍度 | 对 5.3 节结论的修正方向 |
|---------|-----------|----------------------|
| **强连接（核心标的）** | 估值可适度膨胀。市场愿意提前定价 1-3 年甚至更长的预期。 | 5.3 判定"隐含预期 > 实际能力"时，**调降一档警示**——市场认可的高估值不必然是泡沫，是叙事溢价的体现。但要明确这个溢价的可持续性依赖主线本身。 |
| **弱连接（边缘受益）** | 估值可阶段性扩张但缺乏持续性。 | 估值溢价视为"可消失的奖励"，不应作为核心买入理由。如果 5.3 已显示估值偏高，警示级别保持。 |
| **无连接（脱离主线）** | 估值必须由业绩和现金流支撑。市场不给预期溢价，反而可能折价。 | 5.3 判定"隐含预期 < 实际能力"（看似低估）时，**追问是否长期就该如此**——脱离主线的公司，估值修复可能要等到业绩超预期或主线轮动到自己。**估值要谨慎，不要把脱离主线的低估当作机会**。 |

### 5.5.3 主线判断本身的风险

- **主线轮换风险**：A股主线切换迅速。原核心标的会被快速杀估值，回归基本面定价。投资强连接标的实际是在押注主线的持续性。
- **伪主线风险**：短期热点不等于真主线。判断时要看持续度（一般至少持续数月、有产业/政策基本面支撑）和资金共识度，而非单日热度或自媒体噪音。
- **主线判断的主观性**：主线判断带有主观性。在结论中必须明确说明判断依据，并把"主线归属"作为一个**可被推翻的假设**而非事实陈述。
- **置信度标注**：每次主线归属判断都要标注置信度（高/中/低），并说明依据。

### 5.5.4 写作要求

完整基本面研报中，主线归属判断必须作为估值章节的关键变量明确写出，不能跳过。措辞示例：

- "本股票当前与 [X 主线]（[简要说明为何是当前主线]）形成**强连接**，市场对其给予较高估值容忍度。当前 PE 看似偏高，但在主线内属于可接受的预期溢价。主线判断置信度：[高/中/低]，依据：[资金/研报/政策/产业事实]。"
- "本股票当前**不在市场主线内**，估值定价以现实业绩为锚。即使 PE 看似不高，也需要警惕长期'估值修复缺乏催化'的可能性。"
- "本股票与 [X 主线] 形成**弱连接**，主线热度上升时可能享受阶段性溢价，但不构成核心持有理由。"

## 5.6 估值结论（按"六项必输出"收口）

综合 5.1（双轴分型）、5.2（按主锚的快照与历史分位）、5.3（隐含预期）、5.4（三情景）、5.5（主线归属）的分析做最终收口。**主线归属是关键修饰语**——同样的估值水平在主线内和主线外含义完全不同。

### 5.6.A 估值阶段定位

| 估值阶段 | 特征 | 投资者关注点 |
|---------|------|------------|
| **深度低估** | 主锚远低于历史均值和同业中位，市场隐含负增长，但基本面未实质恶化。**主线外此判断更可靠**（因为没有主线踩踏导致的虚假低估）。 | 关注预期差修复的催化剂；主线外标的需要更长的兑现周期 |
| **合理偏低** | 主锚略低于中枢，隐含预期保守，成长性尚可。 | 适合关注超预期的可能性；如果主线轮动至此，可能形成戴维斯双击 |
| **合理定价** | 主锚接近历史中枢与同业中位，隐含预期与实际能力匹配。 | 上行靠业绩超预期或主线加持，下行靠估值回落或主线退潮 |
| **偏高/透支** | 主锚高于历史中枢，隐含高增长预期。**主线内此判断要打折**（高估值是叙事溢价，不必然是泡沫）；**主线外此判断要加重**（缺乏催化的高估更危险）。 | 需要持续高增长兑现，否则有回调风险 |
| **极度乐观** | 主锚极高或无法用盈利/收入/资产合理解释，依赖叙事和远期展望。 | 高波动、高风险，需要主线持续 + 强催化剂双重支撑 |

### 5.6.B 六项必输出（估值结论的最小信息集）

估值结论**不能只写"贵 / 合理 / 便宜"就收口**，必须显式给出以下六项。缺一项就明确写"未取得"，让读者知道缺什么、置信度从哪来。

| # | 必输出项 | 内容要求 | 数据来源 |
|---|---|---|---|
| 1 | **当前估值** | 主锚（PE / PB / PS / 股息率 / EV-EBITDA / EV-ARR 等）+ 数值 + 取数日期；附 1–2 个辅助锚 | `daily-basic`；EV 类指标自算 |
| 2 | **行业中枢** | 行业历史分位 / 行业指数估值中枢，注明窗口；若未取得，明写"未取得，需联网检索申万 / 中证行业指数估值" | 联网检索（脚本无现成接口） |
| 3 | **历史分位** | 公司自身近 1 / 3 / 5 年的 min / p25 / median / p75 / max 与当前分位（主锚维度） | `fetch valuation-band <code> --years 5` |
| 4 | **合理估值区间** | 一个具体的倍数区间（如 PE 18–28×），依据来自：同业中位、自身 3 年 median、行业中枢三者综合；注明权重分配 | 同业 `fetch daily-basic --limit 1` 自算中位 + valuation-band |
| 5 | **上修条件** | 至少 3 条具体的、可观测的事件 / 数据信号，含量化阈值（如"营收增速连续 2 期超 30%"），分业绩/产业/事件/估值四类 | 5.4 三情景的乐观假设 + 行业方法库 §5 模板 |
| 6 | **下修条件** | 至少 3 条具体的、可观测的事件 / 数据信号，含量化阈值（如"毛利率压缩 5 个百分点"），分业绩/产业/事件/估值四类 | 5.4 三情景的压力假设 + 行业方法库 §5 模板 |

**置信度规则**：
- 1 + 3 必须有，否则不能出估值结论。
- 4（同业可比中位）缺失 → 置信度降一档。
- 2（行业中枢）缺失 → 在结论里明确"行业中枢未取得，结论以公司自身历史 + 同业中位为基础"。
- 5 + 6 若写不出具体阈值，只能说置信度低、需要更多基本面跟踪。
- **成长股**：若**未做成长成功率评估**（5.4.C / `growth_success_rate.md`），视为置信度缺口，降一档；若档位为**"低成功率 / 存疑"**，同样降一档，并在结论里写明"溢价依赖团队兑现，而团队 / 治理记录存疑"。成熟龙头 / 红利价值不受此条约束。

**写作模板**（直接搬进报告"估值结论"段）：

```
主估值锚：[PE / PB / PS / 股息率 / EV-EBITDA / …]（依据：5.1 生命周期 = X，5.1 行业 = Y）
辅助锚：[…]

当前估值：[主锚数值] + [辅助锚数值]，取数日期 [YYYY-MM-DD]
行业中枢：[行业指数估值或行业历史分位，注明窗口；或写"未取得"]
公司自身历史分位：近 3 年 median [x]×、当前分位 [x]%（窗口注明）
同业可比中位：[3–5 家可比公司列表 + 各家主锚 + 中位数 + 取数日期]
合理估值区间：[主锚区间，如 PE 18–28×]，依据：[同业中位 + 自身 3 年 median + 行业中枢的权重分配]
估值阶段：[深度低估 / 合理偏低 / 合理定价 / 偏高透支 / 极度乐观]
主线归属修正：[强 / 弱 / 无连接]，对上述判断的方向性影响
成长成功率档位：[高 / 中性 / 低 或 "成熟龙头/红利不适用"]，对三情景概率权重与 B 型期权计入的影响
概率加权期望市值：[期望市值 X vs 当前市值 Y，偏离 Z%（5.4.C）；成熟龙头/红利不适用]
总体置信度：[高 / 中 / 低]，依据：[六项的完备度 + 数据时效 + 成长成功率档位（成长股）]

上修条件（任一成立则估值锚可上修）：
- [业绩类：含量化阈值]
- [产业类：景气 / 主线信号]
- [事件类：订单 / 客户 / 产品 / 政策]
- [估值类：可比中位上移 / 资金流入]

下修条件（任一成立则估值锚需下修）：
- [业绩类：含量化阈值]
- [产业类：景气见顶 / 主线退潮 / 价格战]
- [事件类：客户流失 / 验证失败 / 监管 / 减持]
- [估值类：可比中位下移 / 资金流出 / 解禁压力]
```

## 5.7 财务健康度交叉验证

财务健康度有两个用途：一是作为**定性的质量底座**（输出写进报告第一部分「成长性与财务质量诊断」），二是作为**估值的置信度调节**（异常项在「估值结论」里下调置信度，无论主线归属如何）。以下指标出现异常时需警惕：

**偿债安全**
- 资产负债率：> 70%（非金融行业）需警惕。
- 流动比率 < 1 且速动比率 < 0.5 → 短期偿债压力。
- 有息负债率持续上升 → 财务杠杆风险积累。

**利润质量验证**
- 经营现金流/净利润长期 < 0.7 → 利润回收能力弱。
- 应收账款增速 > 收入增速 → 可能存在激进确认收入。
- 存货增速 > 收入增速 → 可能面临去库存压力。
- 商誉/净资产 > 30% → 减值风险。

**资本效率**
- ROIC（如可得） vs WACC：ROIC > WACC 说明资本配置创造价值。
- 总资产周转率趋势：下降可能暗示资产利用效率降低或过度投资。
