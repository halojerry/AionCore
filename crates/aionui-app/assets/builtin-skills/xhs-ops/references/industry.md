# 小红书运营助手 - 行业垂直打法

> 不同行业的内容策略和标签差异较大，以下是主要垂直行业的参考打法。

---

## 美妆护肤

### 核心内容类型
- **测评类**（主力）：对比测评、长期跟踪测评
- **教程类**：化妆步骤、护肤routine
- **种草类**：好物推荐、平价替代

### 标签策略
```
流量大词：#美妆 #护肤
垂直词：#测评 #粉底液测评 #油皮护肤
人群词：#学生党 #上班族 #敏感肌
内容词：#平价好物 #无限回购 #护肤分享
```

### 爆款规律
- 测评类笔记收藏率最高（解决选择困难）
- 前后对比图是标配
- "平价替代"永远有流量

### 合规重点
- 不得宣称医疗效果（"治疗痘痘"等）
- 不得使用绝对化用语（"最有效"、"保证"等）
- 涉及功效需有依据

- **教程类**：显高技巧、配色技巧
- **合集类**：衣柜整理、购物清单

### 标签策略
```
流量大词：#穿搭 #每日穿搭
垂直词：#小个子穿搭 #微胖穿搭 #职场穿搭
人群词：#学生党 #上班族 #梨形身材
内容词：#显高显瘦 #通勤穿搭 #春日穿搭
```

### 爆款规律
- 真人出镜 > 挂拍
- 竖版3:4封面效果最好
- "一周穿搭不重样"类合集收藏率高

```
流量大词：#美食 #食谱
垂直词：#快手菜 #减脂餐 #烘焙 #家常菜
场景词：#上班族带饭 #宿舍美食 #懒人食谱
内容词：#一人食 #早餐合集 #简单快手
```

### 爆款规律

## 旅游出行

### 核心内容类型
- **攻略类**（主力）：景点打卡、线路推荐
- **好物类**：旅行必备清单、装备推荐
- **体验类**：酒店体验、民宿推荐

### 标签策略
```
流量大词：#旅游 #旅行
垂直词：#小众景点 #周末游 #亲子游
地点词：#上海 #成都 #日本
内容词：#旅行攻略 #拍照技巧 #出行必备
```

### 爆款规律
- 季节性强（节假日、出游季提前布局）
- 小众景点比大众景点更容易爆
- 拍照技巧类视频收藏率高
