## 笔记结构模板

### 种草笔记
```
开头：点名痛点 → "你是不是也在为XX烦恼..."
中间：产品介绍 → 不吹不黑，真实使用感受
结尾：给出结论 → "适合XX人群，不适合XX人群"
标签：精准垂直 + 热门话题
```

### 教程笔记
```
开头：效果承诺 → "学会这X步，XX天搞定XX"
中间：步骤拆解 → 每步配图或示意图
结尾：效果对比 → Before/After，收藏必备
标签：教程类 + 领域词 + 问题词
```

### 合集笔记
```
开头：收藏理由 → "XX合集，全网最全，建议收藏"
中间：分类整理 → 编号 + 简短描述
结尾：总结推荐 → "最值得买的是XX"
标签：合集 + 领域 + "清单"/"推荐"类词
```

### 人设笔记
```
开头：真实场景 → "今天遇到了XX事..."
中间：内心独白 → 真实想法，不完美但要真诚
结尾：共鸣引导 → "你们有没有遇到过？"
标签：日常 + 情绪词 + 身份词
```

## 爆款分析模板

当某篇笔记数据好时，分析：
1. 哪个因素做对了？（封面？标题？选题？标签？）
2. 这篇笔记的什么特点让用户收藏了？
3. 能否复制这个模式做第二篇？
4. 如果数据不好，是哪个环节出了问题？

## 简介公式

```
[身份] | [核心内容方向]
[成就背书/经历]：让用户信任
[价值承诺]：让用户知道关注能得到什么
```

**案例**：
```
服装设计师 | 小个子穿搭研究僧
专注矮个子显高穿搭，已帮助3000+女生找到风格
关注我，一起变美💃 每周更新
```
